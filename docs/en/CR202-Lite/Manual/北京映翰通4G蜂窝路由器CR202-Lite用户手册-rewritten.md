## 第1章 认识设备

### 1.1 概述

CR202-Lite是一款集4G蜂窝网络、Wi-Fi、VPN技术于一体的移动无线路由器，能够在室内、户外及移动场景下提供稳定可靠的网络连接。该设备支持有线以太网接入，提高了入网的多样性；内置链路检测机制可有效保障网络连续性。部分型号配备内置电池，支持随时随地移动工作。结合InHand Device Manager（DM）云管理平台，CR202-Lite可为客户提供便捷的远程统一管理服务。

#### 硬件方式（Reset键）

1. 设备上电后，立即按下Reset键并保持3~5秒，直到系统指示灯变为黄色常亮。
2. 松开Reset键，系统指示灯变为黄色闪烁。
3. 再次按住Reset键，待系统指示灯变为黄色常亮后松开，随后系统指示灯变为绿色闪烁，表示设备恢复出厂设置成功。

## 第2章 安装与首次使用

### 2.1 安装前准备

#### 环境要求

- 工作温度：-10℃～50℃；存储温度：-20℃～60℃。
- 安装位置应避免阳光直射、远离发热源及强电磁干扰区域。
- 需确认现场在3G/4G网络覆盖范围内，且无信号屏蔽。

#### 工具及物料准备

| 物品 | 说明 |
|------|------|
| PC一台 | 用于配置和访问设备Web管理界面 |
| Nano SIM卡一张 | 已开通数据业务且未欠费停机（蜂窝上网场景） |
| 电源适配器 | 5V/2A Type-C供电（可使用附带的USB转Type-C线） |
| 网线 | 用于连接PC与设备LAN口 |
| 膨胀螺丝 | 用于壁挂安装（如需要） |

> **注意**：非电池型号需连接外部电源；带电池型号支持电池供电，也可使用Type-C线充电。设备的安装操作（如插拔SIM卡）需在断电状态下进行。

### 2.2 安装指南

#### SIM卡安装

CR202-Lite支持单Nano SIM卡或eSIM。安装Nano SIM卡时，需拆开后盖，取出电池（如适用），按图示位置放入SIM卡。

<p align="center"><img src="./img/3TdbKst1UzjRJIgj/1728453187808-f70b6b36-19a8-441f-aeee-bae1d8dfe56a-053263.png" alt="SIM卡安装位置"></p>

<p align="center"><strong>图 2-1 SIM卡安装位置</strong></p>

> **警告**：更换或插拔SIM卡时，必须先将设备断电，以免造成数据丢失或设备损坏。

#### 供电安装

非电池型号需将5V/2A Type-C电源线连接至设备，观察系统指示灯是否点亮。若指示灯未亮，应检查电源连接或联系技术支持。

带电池型号支持按压开关按钮3秒开机，也可连接5V/2A Type-C电源进行充电和使用。

> **警告**：
> - 禁止在高温环境（如阳光暴晒下或高温车内）使用或放置设备，否则可能导致电池过热、起火或功能失效。
> - 严禁将设备置于海水或水中；保存不用时，应放置在阴凉干燥的环境中。

#### 壁挂安装

CR202-Lite可水平放置于桌面，也可使用膨胀螺丝进行壁挂安装。安装时需避开强热源及电磁干扰源。

#### 网络连接与PC配置

完成硬件安装后，通过网线将PC与CR202-Lite的LAN口相连。

**方法一：DHCP自动获取地址（推荐）**

将PC的以太网适配器设置为"自动获得IP地址"和"自动获得DNS服务器地址"（操作系统默认配置），由设备自动为PC分配IP地址。

将PC的IP地址设置为与设备LAN口同一网段，例如：
- IP地址：192.168.2.2
- 子网掩码：255.255.255.0
- 默认网关：192.168.2.1
- DNS：8.8.8.8或运营商DNS服务器地址

## 第3章 常用场景配置

### 场景1：蜂窝联网

**目标**：通过4G蜂窝网络接入互联网。

**前提**：已插入Nano SIM卡并安装Cellular天线，设备已上电；SIM卡已开通数据业务且未欠费停机。

**预计用时**：约5分钟。

**操作步骤**：
1. 在设备下电状态下插入Nano SIM卡，将3G/4G LTE天线安装至Cellular天线接口，随后连接电源线并接入网线（用于连接配置PC）。
2. 使用浏览器登录设备Web管理界面（默认地址 `192.168.2.1`）。
3. 点击【网络】→【Cellular】菜单。设备默认开启蜂窝拨号功能，通常等待几分钟后即可自动拨号上网；若未自动拨号，可检查"启用"开关状态并重新启用。

   <p align="center"><img src="./img/yldgHRB6CkrlXBp4/1728355392497-2cafa9bc-270c-4d1c-bd8f-35c91b01ca68-299517.png" alt="SIM卡拨号配置界面"></p>

   <p align="center"><strong>图 3-1 SIM卡拨号配置界面</strong></p>

4. 点击【状态】→【网络连接】查看拨号端口状态，确认显示"已连接"且已获得分配的IP地址、DNS等信息后，表明蜂窝拨号成功。

**验证方法**：
1. 观察设备联网指示灯，确认其变为绿色常亮（无电池版本）或按带电池版本的拨号成功状态显示。
2. 在【工具】→【PING探测】中，输入外部地址（如 `8.8.8.8`）进行连通性测试，确认可正常通信。

**常见问题**：
- 无法拨号：检查SIM卡是否插反、未插紧或接触不良；确认SIM卡已开通数据业务且未欠费；必要时向运营商核对APN参数。
- 信号弱或无服务：检查Cellular天线是否安装牢固；将设备移至信号覆盖更好的位置。

### 场景2：有线联网

**目标**：通过有线WAN口接入互联网。

**前提**：WAN口已连接可用的外网网线，LAN口已连接配置PC，设备已上电；若设备不使用SIM卡拨号上网，需先在【网络】→【Cellular】中禁用拨号接口，避免设备因循环拨号达到最大次数后自动重启。

**预计用时**：约5分钟。

**操作步骤**：
1. 连接电源线，并将外网网线接入设备WAN口，将PC网线接入设备LAN口。
2. 配置PC的IP地址与设备处于同一网段（推荐DHCP自动获取；手动设置时IP地址可为 `192.168.2.2`~`192.168.2.254`，网关为 `192.168.2.1`，子网掩码为 `255.255.255.0`）。
3. 打开浏览器，输入 `192.168.2.1` 进入Web管理界面并登录。
4. 点击【网络】→【WAN】菜单，创建WAN口并选择地址获取方式：
   - **动态DHCP（推荐）**：由上级网络设备自动分配IP地址。

     <p align="center"><img src="./img/yldgHRB6CkrlXBp4/1728355391172-452a0334-daec-4d44-9e04-37a30619db76-101150.png" alt="动态DHCP获取WAN口地址"></p>

     <p align="center"><strong>图 3-2 动态DHCP获取WAN口地址</strong></p>

   - **静态地址**：手动填写IP地址、子网掩码、网关及DNS后点击"应用"。

     <p align="center"><img src="./img/yldgHRB6CkrlXBp4/1728355391395-206eaab5-9b40-4ab1-a738-c74dd2132802-373892.png" alt="静态IP配置WAN口地址"></p>

     <p align="center"><strong>图 3-3 静态IP配置WAN口地址</strong></p>

   - **ADSL拨号（PPPoE）**：填写运营商提供的拨号用户名和密码后点击"应用"。

     <p align="center"><img src="./img/yldgHRB6CkrlXBp4/1728355391616-d4c430f9-d85c-4a4b-8e61-0febe684ff65-448173.png" alt="ADSL拨号上网配置"></p>

     <p align="center"><strong>图 3-4 ADSL拨号上网配置</strong></p>

5. 配置完成后，点击【工具】→【PING探测】，输入目标地址测试网络连通性。

   <p align="center"><img src="./img/yldgHRB6CkrlXBp4/1728355391909-8195f3e0-45eb-47e7-ad16-a1d1e8b44818-947707.png" alt="网络连通性验证"></p>

   <p align="center"><strong>图 3-5 网络连通性验证</strong></p>

**验证方法**：
1. 进入【状态】→【网络连接】，确认WAN口显示"已连接"且已获得IP地址。
2. PING探测结果正常，可访问互联网。

**常见问题**：
- 无法获取IP地址：检查WAN口网线是否可靠连接到外网；确认上级路由器或光猫工作正常。
- 无法上网：检查WAN配置中"默认路由"及"共享连接（NAT）"是否已启用；检查DNS设置是否正确。

### 场景3：Wi-Fi AP模式配置

**目标**：将设备配置为无线接入点（AP），为周边终端提供Wi-Fi接入服务。

**前提**：设备本身已通过有线或蜂窝方式接入互联网；管理PC已通过网线或Wi-Fi连接到设备。

**预计用时**：约5分钟。

**操作步骤**：
1. 登录设备Web管理界面。
2. 点击【网络】→【WLAN（AP模式）】菜单。
3. 启用WLAN功能，选择工作模式（如802.11b/g/n），设置SSID广播、SSID名称、认证方式及加密方式。
4. 在"密钥"处设置无线终端连接时所需的密码。

   <p align="center"><img src="./img/yldgHRB6CkrlXBp4/1728355392796-156ee091-8133-4f9b-8bc0-325a7a77f684-264369.png" alt="AP模式配置界面"></p>

   <p align="center"><strong>图 3-6 AP模式配置界面</strong></p>

5. 点击"保存"或"应用"使配置生效。

**验证方法**：
1. 使用其他无线终端扫描并连接所配置的SSID。
2. 终端获取到IP地址后，确认可正常访问互联网。

**常见问题**：
- 终端搜索不到SSID：检查"SSID广播"是否已开启；确认WLAN功能已启用。
- 终端连接后无法上网：确认设备本身已通过WAN或蜂窝链路接入互联网；检查认证密码是否输入正确。

### 4.1 系统设置

#### 4.1.1 基本设置

该页面用于设置路由器Web配置界面的显示语言及主机名称。点击【系统】→【基本设置】菜单，进入【基本配置】页面进行设置。

| 参数名称 | 说明 | 缺省值 |
|----------|------|--------|
| 界面语言 | Web配置页面的显示语言 | 中文 |
| 主机名 | 路由器的标识名称，便于管理识别 | Router |

#### 4.1.2 系统时间

系统时间配置用于确保路由器与其他设备协调工作。通过配置时区和NTP自动更新时间，可实现网络内设备的时钟同步。点击【系统】→【系统时间】菜单，进入【系统时间】界面。在主机时间部分点击"同步时间"按钮，可将连接PC的当前时间设置为路由器的系统时间。

| 参数名称 | 说明 | 缺省值 |
|----------|------|--------|
| 路由器时间 | 显示路由器当前系统时间 | 2021-xx-xx 08:00:00 |
| 主机时间 | 显示当前连接PC的时间 | 当前时间 |
| 时区 | 设置路由器所在地理时区 | 定制 |
| 设置时区字符串 | 设置路由器所在时区字符串 | CST-8 |
| 自动更新时间 | 选择是否自动更新时间，可选启动时、每1/2/…小时等 | 启动时 |
| 网络时间服务器（NTP） | 设备联网后连接该NTP服务器同步时间 | 1.pool.ntp.org |

#### 4.1.3 管理控制

管理控制提供HTTP、HTTPS、TELNET和SSHD四种远程及本地管理服务。

- **HTTP**：超文本传输协议，用于在Internet上传递Web页面信息。启用后可通过HTTP协议登录设备Web界面。
- **HTTPS**：HTTP的安全版本，支持SSL协议，可提高访问安全性。
- **TELNET**：提供远程登录和虚拟终端功能。设备支持Telnet客户端和Telnet服务器模式。
- **SSHD**：允许其他设备使用SSH协议远程访问设备终端。

点击【系统】→【管理控制】菜单，进入【管理控制】界面进行配置。

| 参数名称 | 说明 | 缺省值 |
|----------|------|--------|
| **用户名/密码** | | |
| 用户名 | 设置登录Web配置界面的用户名 | adm |
| 旧密码 | 原Web登录密码 | 空 |
| 新密码 | 设置新密码，长度至少10个字符，且包含大写字母、小写字母、数字、特殊字符中任意三种 | 空 |
| 确认密码 | 再次确认新密码 | 空 |
| **管理功能** | | |
| 服务端口 | HTTP/HTTPS/TELNET/SSHD的服务端口 | 80/443/23/22 |
| 本地管理 | 启用—允许本地局域网使用相应服务管理路由器；禁用—不允许 | 启用 |
| 远程管理 | 启用—允许远程主机使用相应服务管理路由器；禁用—不允许 | 启用 |
| 允许远程管理的地址范围 | 设置允许远程管理的主机地址范围（仅限HTTP/HTTPS/TELNET） | 空 |
| 说明 | 记录管理功能参数的意义（不影响配置） | 空 |
| **控制台登录用户** | 设置TELNET/SSHD登录的用户名和密码（需点击"新增"保存） | |
| 用户名 | 配置控制登录用户名，用户自定义 | 无 |
| 密码 | 配置控制登录密码，用户自定义 | 无 |
| **其他参数** | | |
| 登录超时 | 设置登录超时自动断开时间 | 500秒 |

![管理控制提示](./img/3TdbKst1UzjRJIgj/1728354473841-37f3046a-8355-4e6e-8e9c-fd5ec65f06b6-307648.png)
- "用户名/密码"配置部分可更改Web登录用户名和密码，但不能新建用户名，即Web登录方式仅支持该单一用户名。

#### 4.1.7 系统升级

系统升级共分为两个阶段：第一阶段将升级文件写入备份固件区；第二阶段在系统重启时将备份固件拷贝到主固件区。升级过程中，请勿在Web界面进行任何操作，否则可能导致升级中断。

点击【系统】→【系统升级】菜单，进入【系统升级】界面。升级步骤如下：

### 案例1：分支机构蜂窝接入与远程管理

**场景描述**

小型分支机构或自助服务终端通过CR202-Lite的4G蜂窝网络接入互联网，为现场PC及无线终端提供网络服务。同时，设备接入InHand Device Manager（DM）云平台，实现设备的远程状态监控、统一配置管理及故障告警。

**设备角色**

本设备作为分支机构的边缘接入网关，负责通过4G蜂窝网络建立互联网连接，并为本地有线及无线终端提供NAT转发、DHCP分配及Wi-Fi接入服务；同时作为DM平台的被管设备，接受远程统一运维。

**配置步骤**

1. **硬件安装**：安装Nano SIM卡及Cellular天线，连接Type-C电源，将PC网线接入设备LAN口。
2. **首次登录**：按[安装与首次使用](#第2章-安装与首次使用)完成PC网络设置，通过浏览器访问 `192.168.2.1` 登录Web管理界面。
3. **蜂窝联网**：按[场景1：蜂窝联网](#场景1蜂窝联网)确认蜂窝拨号正常，设备可访问互联网。
4. **本地Wi-Fi接入（如需要）**：按[场景3：Wi-Fi AP模式配置](#场景3wi-fi-ap模式配置)启用AP模式，配置SSID及密码，供无线终端接入。
5. **接入DM平台**：按[场景5：设备远程管理平台接入](#场景5设备远程管理平台接入)配置服务器地址、注册账户及序列号，完成平台注册与设备添加。
6. **远程验证**：在DM平台查看设备在线状态，测试远程下发配置或重启功能。

## 附录 安全注意事项

1. 设备应在规定的工作温度（-10℃～50℃）和存储温度（-20℃～60℃）范围内使用。
2. 请勿在易燃易爆环境中使用本设备。
3. 电源连接前请确认电压符合设备规格要求（5V/2A Type-C）。
4. 设备的安装及SIM卡插拔操作必须在断电状态下进行，以防数据丢失或设备损坏。
5. 壁挂安装时应避开阳光直射、热源及强电磁干扰区域。
6. 建议使用已有网络知识的人员进行配置，首次安装应在合格工程师指导下进行。

> **警告**：非专业人员请勿擅自打开设备外壳，以免发生触电或其他危险。

### 带电池型号特别注意事项

1. 禁止将电池置于高温热源（如火源、加热器）旁使用和留置。
2. 禁止将设备放置在太阳光直射的地方。
3. 禁止在高温环境（如阳光暴晒下或高温车内）使用或放置设备，否则可能引起电池过热、起火或功能失效。
4. 禁止将电芯短路、过充或过放。
5. 严禁将设备浸入海水或水中；保存不用时，应放置在阴凉干燥的环境中。
6. 严禁颠倒正负极使用电池。
7. 不要拆卸或修整电芯。
8. 禁止将电池与金属物品（如发夹、项链等）一起运输或储存。
9. 不要使电芯受到明显的损害或变形。
10. 严禁将电芯直接插入电源插座。
11. 如果电芯发生泄漏，电解液进入眼睛，不要搓揉，应用清水冲洗眼睛，并立即前往医院接受治疗。
12. 禁止与液态锂离子或不同型号的聚合物锂电池混合使用。
13. 电池应放置在儿童接触不到的地方。
14. 禁止直接焊接电池极耳，禁止用钉子或其他利器刺穿电池。
15. 禁止敲击、抛掷或踩踏电池。
16. 充电时请选用锂离子电池专用充电器。
17. 如果电池发出异味、发热、变色、变形，或在充电、使用、储存过程中出现任何异常，应立即将电池从设备或充电器中移离并停用。
18. 更换电芯应由电芯供应商或设备供应商完成，用户不得自行更换。
19. 废弃电池应用绝缘纸包住电极，以防起火或爆炸。
20. 禁止使用已损坏的电芯。
