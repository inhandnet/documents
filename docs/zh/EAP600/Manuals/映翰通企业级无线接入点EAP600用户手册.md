## 第1章 认识设备

### 1.1 概述

EAP600是北京映翰通网络技术股份有限公司面向商业领域推出的Wi-Fi 6无线接入点（AP）产品。该产品可为小型商业场所、企业分支机构和酒店等需要无线覆盖的场景提供高速、安全的无线网络接入。EAP600支持Wi-Fi 6标准，具备强大的网络接入能力、全面的安全特性以及智能化的软件服务，可实现高效的网络管理与安全无忧的数据传输。

## 第2章 安装与首次使用

### 2.1 安装前准备

#### 环境要求

| 项目 | 要求 |
|------|------|
| 工作温度 | 0℃ ~ 45℃ |
| 存储温度 | -40℃ ~ 70℃ |
| 安装位置 | 避免阳光直射，远离热源或强电磁干扰源 |
| 承重条件 | 安装位置需足够坚固，能够支撑设备及安装配件重量 |

#### 工具与配件准备

- 以太网线（建议使用CAT5e及以上规格）
- 支持IEEE 802.3 af/at标准的PoE供电设备，或原装12V DC电源适配器
- 吸顶安装所需的膨胀螺钉及钻孔工具

> **警告**：需使用符合IEEE 802.3 af/at标准的PoE供电设备为AP提供电源。若不使用PoE方式供电，应使用原装电源适配器，以免损害设备。

### 2.2 安装指南

#### PoE供电安装

EAP600支持IEEE 802.3 af/at标准受电，仅用一根以太网线即可完成对设备的供电及数据传输。

1. 将以太网线缆的水晶头插入设备上的RJ-45接口。
2. 将以太网线的另一端接入符合802.3 af/at标准的PoE设备。
3. 观察设备面板电源指示灯，确认设备正常上电。

<p align="center"><img src="./img/QKBagCyG6U7Quzyd/1721109731787-20af82ee-3f8c-4d2c-b5db-26ded235587c-221375.webp" alt="设备通过PoE接入电源"></p>

#### 电源适配器供电

EAP600支持选配12V DC电源适配器供电。使用电源适配器时，应确保设备安装位置周围有电源插座。

1. 将电源适配器圆孔插头一端插入设备电源插孔。
2. 将电源适配器另一端插入电源插座。

<p align="center"><img src="./img/QKBagCyG6U7Quzyd/1721109732010-15d68a83-3a54-4126-bb99-f8ae345b732f-176027.webp" alt="设备通过电源适配器接入电源"></p>

1. 使用钻头或适当工具在标记位置预钻孔，确保开孔尺寸与膨胀螺钉匹配。
2. 将螺钉穿过吸顶支架并固定到天花板开孔中，确保支架牢固安装。

### 2.3 Web界面登录

1. 将PC通过Wi-Fi连接到EAP600（默认SSID及密码参考[默认设置](#16-默认设置)）。
2. 确认PC已获取到属于`192.168.10.0`网段的IP地址（设备默认开启DHCP Server功能）。
3. 在浏览器地址栏中输入设备默认管理地址`192.168.10.1`。
4. 输入用户名`adm`和密码`123456`，进入设备Web管理界面。
5. 若浏览器提示网页不安全，需点击"高级"或"隐藏"，选择"继续前往"。

### 2.4 快速检查

安装及初始化完成后，建议按以下清单进行检查：

- [ ] 以太网线已正确连接至WAN口
- [ ] PoE设备正常工作且对应端口PoE功能已开启，或电源适配器正常供电
- [ ] 吸顶安装件已牢固固定
- [ ] PWR指示灯状态正常（绿色闪烁后常亮或快闪表示正常工作）
- [ ] PC可通过Wi-Fi连接至设备并获取到`192.168.10.0`网段地址
- [ ] 可通过浏览器正常访问`192.168.10.1`管理界面

## 第3章 常用场景配置

### 场景1：WAN口联网配置

**目标**：使EAP600通过有线WAN口接入互联网。

**前提**：设备已上电，WAN口已连接至上行网络设备（路由器/交换机），PC已通过Wi-Fi连接至EAP600并可访问Web管理界面。

**预计用时**：约5分钟。

**操作步骤**：
1. 登录设备Web管理界面。
2. 进入【配置】→【WAN】菜单。
3. 选择网络接入方式：
   - **DHCP**（默认）：设备自动从上行DHCP Server获取IP地址。
   - **静态IP**：手动输入从网络管理员处获取的IP地址、子网掩码、网关及DNS服务器地址。MTU值缺省为1500，输入范围为128~1500之间的整数。
4. 点击"保存"按钮，等待配置生效。

<p align="center"><strong>图 3-2 配置静态IP接入网络</strong></p>

**验证方法**：
1. 进入【仪表盘】→【接口状态】，确认WAN接口图标变为绿色。
2. 点击WAN端口图标，查看工作模式、IP地址、DNS服务器等信息是否正确。
3. 使用【系统】→【工具】→【Ping】功能，Ping公共DNS地址（如`8.8.8.8`），确认网络连通。

**常见问题**：
- 无法获取IP地址：检查网线连接是否牢固，确认上行设备DHCP服务是否正常开启。
- 静态IP配置后无法上网：核对IP地址、子网掩码、网关及DNS参数是否与网络管理员提供的信息一致。

### 场景2：Wi-Fi接入点配置

**目标**：配置EAP600的无线网络参数，为终端设备提供Wi-Fi接入服务。

**前提**：已通过PC或云平台登录设备管理界面。

**预计用时**：约5分钟。

**操作步骤**：
1. 进入【Wi-Fi】→【SSIDs】菜单。
2. 点击"添加"或已有SSID的"编辑"按钮。
3. 配置SSID名称、频段（2.4GHz/5GHz）、认证方式及密码等参数。
   - 设备默认配置2.4G和5G两个主SSID，主SSID频段不支持修改且不支持删除。
   - 已添加的SSID频段不支持修改，信道将自动与所属主SSID的信道保持一致。
4. 点击"保存"按钮，使配置生效。

**验证方法**：
1. 使用手机或电脑搜索配置的SSID名称。
2. 使用配置的密码连接Wi-Fi，确认能够正常获取IP地址并访问互联网。

**常见问题**：
- 终端无法搜索到SSID：检查对应频段的射频是否已启用，确认SSID未设置为隐藏。
- 连接后无法上网：检查WAN口联网配置是否正确，确认SSID绑定的子网类型设置无误。

**场景描述**：在小型商业场所（如零售门店、咖啡厅）中部署EAP600，通过有线WAN口接入互联网，为店内顾客和员工提供稳定的高速Wi-Fi覆盖。

**设备角色**：EAP600作为无线接入点，负责将来自上层路由器的有线网络信号转换为Wi-Fi信号，为无线终端提供网络接入服务。

**配置步骤**：
1. 参照[第2章 安装与首次使用](#第2章-安装与首次使用)完成EAP600的上电和安装。

3. 参照[场景2：Wi-Fi接入点配置](#场景2wi-fi接入点配置)，配置2.4GHz和5GHz的SSID名称及密码。
4. 根据需要，在"配置"→"LAN"中创建访客网络，实现顾客网络与内部网络的隔离。
5. 如需远程管理，参照[场景3：接入小星云管家云平台](#场景3接入小星云管家云平台)将设备添加到云平台。

## 附录 安全注意事项

1. 设备应在规定的温度和湿度范围内使用（工作温度：0℃ ~ 45℃）。
2. 请勿在易燃易爆环境中使用本设备。
3. 需使用符合IEEE 802.3 af/at标准的PoE供电设备为AP提供电源。若不使用PoE方式供电，应使用原装电源适配器，以免损害设备。
4. 请勿将设备安装在强电磁干扰环境中，避免将其放置在高功率设备附近。
5. 安装完成后，需确认设备稳固固定，以防设备摔落造成设备损坏或人身伤害。
6. 需定期检查设备线缆，保证线材清洁，如有破损应及时更换。
7. 清洁设备时，请勿将化学试剂直接喷洒在设备表面，避免损害外壳或内部组件。需使用柔性布料清洁。
8. 请勿私自拆修或改装设备，否则将导致安全问题且设备会失去保修资格。

> **警告**：非专业人员请勿打开设备外壳，存在触电风险。

## FAQ 常见问题解答

### 问题1：云平台是否免费？

用户使用云平台服务时，需逐台购买许可费用以享受丰富的云端功能。设备首次添加到平台时，默认赠送1年免费基础版许可，后续可按需续费。

### 问题2：设备如何添加到云平台？

1. 通过[https://star.inhandcloud.cn/](https://star.inhandcloud.cn/)网址注册小星云管家登录账号。
2. 使用注册账号登录云平台，在【设备】菜单下点击"添加"，按照提示填写设备的序列号和MAC地址，即可完成添加。

### 问题3：设备是否可以不用云平台？

可以。用户可在本地完成绝大多数配置功能，但批量配置下发、升级、SD-WAN、Connector等功能需结合云平台使用。
