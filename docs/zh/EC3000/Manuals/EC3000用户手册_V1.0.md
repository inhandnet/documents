## 产品信息

### 产品简介

EC3000产品系列搭载RK3588平台，提供Hailo-8 AI算力扩展，是边缘AI运算应用的理想选择。系统提供2个RS-232串行接口、2个RS-485串行接口、1个CAN局域网控制器、4个数字量输入与4个数字量输出、1个网络指示灯、1个STATUS系统状态灯、1个USER用户灯、1个PWR电源灯、2个网络接口、1个USB Type-C 3.0 OTG、4个USB 3.0、2个HDMI用户连接视频输出、1个麦克风接口、1个SPK功率输出接口、1个SIM卡槽支持2个Nano卡、1个RESET系统重置按键、1个ON/OFF开关机按键。

### 接口

#### 指示灯

![1726621040998-f55a7865-241e-4394-bedc-901027232312.png](./img/HQKzvb71mHvtE9El/1726621040998-f55a7865-241e-4394-bedc-901027232312-976679.png)

前面板设有4个LED，分别是：

1. PWR电源开/关指示灯（红色 LED），该灯可指示系统开机状态，当PWR LED亮起表示系统处理开机状态，PWR LED熄灭表示系统处于关机状态；
2. STATUS系统运行状态指示灯（绿色 LED），该灯可指示系统运行状态，当STATUS LED闪烁（频率1Hz）时表示系统运行正常，STATUS LED熄灭表示系统未运行；
3. 4G/5G网络状态指示灯（绿色 LED），该灯可指示蜂窝网络连接状态，当4G/5G LED亮起时表示蜂窝网络连接成功，4G/5G LED闪烁表示蜂窝网络未连接；
4. USER可编程灯（绿色 LED），该灯可由用户编程实现LED状态逻辑。

5. 建议电源功率 ≥ 36W。

### 账户管理

#### 创建账户

点击Show Applications->Terminal或鼠标右键点击Open in Terminal，并输入下述命令，根据提示输入密码以及账户信息，创建前请确认账户是否存在，对于已经存在的账户，再次创建将提示The user 'username' already exists 。

```json
## 查询test账户是否存在
id test
## 创建test账户
sudo adduser test
```

![1725427942525-3b250bd8-0bd4-45c0-9640-da518e201c13.png](./img/HQKzvb71mHvtE9El/1725427942525-3b250bd8-0bd4-45c0-9640-da518e201c13-903484.webp)

#### 删除账户

点击Show Applications->Terminal或鼠标右键点击Open in Terminal，并输入下述命令删除账户，删除账户前请确认账户是否存在，如果删除不存在的账户将提示The user 'username' does not exist。

```json
## 查询test账户是否存在
id test
## 删除test账户
sudo deluser test
```

![1725427988846-a5a1a4f4-59ef-4229-b2fe-b6e98d48a614.png](./img/HQKzvb71mHvtE9El/1725427988846-a5a1a4f4-59ef-4229-b2fe-b6e98d48a614-735171.png)

#### 禁用和启用账户

点击Show Applications->Terminal或鼠标右键点击Open in Terminal，并输入下述命令禁用/启用账户，禁用/启用账户前请确认账户是否存在，如果账户不存在将提示The user 'username' does not exist。

```json
## 查询test账户是否存在
id test
## 禁用test账户
sudo passwd -l test
## 启用test账户
sudo passwd -u test
## 查询test账户状态（L禁用/P启用）
```

### 网络设置

#### 基于IEOS的web管理

- EC3000基于Ubuntu22.04系统，因此可以使用Linux原生的命令进行网络管理和系统管理；为了便于用户配置，映翰通开发了一套IEOS系统程序，提供web界面，用户可以方便的通过web进行网络管理和系统管理，但需要注意的是，当启用IEOS功能后，IEOS会接管网络管理和系统管理，此时通过Linux原生命令进行网络管理和系统管理可能会失效；设备出厂默认启用IEOS，如果用户需要基于Linux原生的命令行进行网络管理和系统管理，需要先关闭IEOS。
- IEOS是映翰通自研的一套运行在Linux系统上的网络管理和系统管理程序，IEOS提供web界面，用户可以通过web进行以太网口ip地址，蜂窝拨号，Wi-Fi Station，DHCP Client/Server，静态路由，防火墙等网络配置；也可以对系统时间，时区，固件升级和系统重启等进行操作；另外IEOS还支持对接映翰通的设备管理平台DeviceLive，用户可以通过DeviceLive平台对EC3000设备进行远程监控和管理。
- IEOS采用状态和配置分离的设计方案，分为网络管理，系统管理和状态三大功能板块。网络管理菜单和系统管理菜单下只能进行网络和系统的相关配置，状态信息需要统一到状态页面查看。
- 重要说明：使用IEOS程序管理网络配置和系统配置的时候，如果同时使用Linux原生命令，二者可能会相互影响，导致异常的运行状态。建议IEOS支持的配置都通过IEOS web来实现管理，IEOS不支持的配置，比如VPN，可以结合Linux原生命令一起实现配置目标。
- 使用Linux命令行进行网络配置和系统配置的时候，首先需要关闭IEOS程序。IEOS是通过systemctl来管理的，关闭IEOS的方式如下：

```json
systemctl disable ieos_daemon
reboot
```

- 重要说明：关闭IEOS之后，拨号，Wi-Fi等无线联网功能需要用户基于Linux原生命令来实现，而且也无法实现对接DeviceLive平台对设备进行远程管理。

### 系统重置

#### 页面重置

点击系统管理（System）->其他（Others），选择系统重置（System Reset）->重置（Reset）。

![1725878836099-53397b86-cc18-4ea3-847c-51636c134837.png](./img/HQKzvb71mHvtE9El/1725878836099-53397b86-cc18-4ea3-847c-51636c134837-952152.png)

#### 按键重置

- 设备在右面板有1个RESET针孔按键，在系统正常运行时按住RESET 10秒等待系统状态灯由闪烁变为常亮并松开，设备将进入系统重置状态（恢复为系统出厂状态）。
- 也可以在上电之前按住RESET键，然后上电，持续按住约5秒，直到Status灯长亮后5秒内松开，设备将进入系统重置状态（恢复为系统出厂状态）。
- 此功能可以使用命令禁用或启用：`sudo update resetkey <enable/disable>`
- 使用此命令：`sudo update resetkey`可以查看是否启用（默认启用）

#### 命令重置

打开Terminal，使用update命令可以执行系统重置。

```shell
sudo update reset
```
