## 第1章 认识设备

### 1.1 概述

ER815是北京映翰通面向商业联网领域推出的新一代高性能Wi-Fi6路由器，具备1Gbps的转发能力和多网口设计，支持300Mbps VPN能力。通过搭配分支站点的ER815，可实现分支间组网。通过小星云平台，IT管理人员可了解每个门店的网络状态、流量使用、客户端、实时告警等信息，降低IT管理难度。

## 第2章 安装与首次使用

### 2.1 安装前准备

**环境要求**：

| 项目 | 要求 |
|------|------|
| 温度 | 设备应在规格书要求的温度范围内使用 |
| 湿度 | 设备应在规格书要求的湿度范围内使用 |
| 电源 | 使用原装电源适配器，避免因电源适配器和设备不匹配造成设备损坏 |
| 安装位置 | 避免强电磁干扰环境，避免与高功率设备过于接近 |

> **注意**：安装完成后需确认设备是否易发生晃动，避免因设备跌落造成设备损坏。

**工具准备**：

- 网线（RJ45）
- 电脑（用于配置）
- SIM卡（蜂窝模块使用，可选）
- 蜂窝天线（蜂窝模块使用，可选）

### 2.2 安装指南

#### 2.2.1 设备连接

1. 将设备接入电源，并用网线将设备任意LAN口连接至PC
2. 如需使用蜂窝网络，插入SIM卡并安装天线
3. 如需通过有线网络接入，用网线连接WAN口至上级网络

#### 2.2.2 PC网络配置

设备默认开启DHCP服务，PC可设置为"自动获得IP地址"，边缘路由器将为PC自动分配地址。需确认PC获取的IP地址处于192.168.2.0网段。

若PC无法自动获取IP地址，需手动配置：

| 配置项 | 值 |
|--------|-----|
| IP地址 | 192.168.2.2~192.168.2.254中的任意地址 |
| 子网掩码 | 255.255.255.0 |
| 默认网关 | 192.168.2.1 |
| DNS | 223.5.5.5或运营商DNS服务器地址 |

<p align="center"><strong>图 2-1 PC手动配置/自动获取IP地址</strong></p>

#### 2.2.3 Web界面登录

1. 在浏览器地址栏中输入设备默认地址 `192.168.2.1`
2. 输入用户名和密码（参考铭牌信息，默认用户名为 `adm`）
3. 如页面提示"网页不安全"，点击"高级"，选择"继续前往"

安装完成后，按以下清单进行检查：

- [ ] 设备电源指示灯正常亮起
- [ ] PC与设备LAN口连接正常，PC获取到IP地址

## 第3章 常用场景配置

### 场景1：有线联网

**目标**：通过有线网络（DHCP、静态IP或PPPoE）接入互联网。

**前提**：已用网线将设备WAN口连接至上级网络，设备已上电。

**预计用时**：约5分钟。

**操作步骤**：

1. 登录设备Web管理界面
2. 进入【Internet】→【WAN1】菜单（或【WAN2】）
3. 根据网络环境选择连接方式：
   - **DHCP**：默认配置，自动获取IP地址，无需额外配置
   - **静态IP**：手动填写从运营商或网络管理员获取的IP地址、子网掩码、网关、DNS
   - **PPPoE**：填写宽带拨号账号和密码（由运营商提供）
4. 点击"保存"按钮

<p align="center"><strong>图 3-1 设备WAN口配置</strong></p>

**验证方法**：

1. 查看【状态】→【链路监控】，确认WAN接口状态为"已连接"
2. 使用设备自带的Ping工具测试互联网连通性

### 场景2：蜂窝联网

**目标**：通过4G/5G蜂窝网络接入互联网。

**前提**：已插入SIM卡并安装天线，设备已上电，设备支持蜂窝模块。

**预计用时**：约5分钟。

**操作步骤**：

1. 将SIM卡插入卡槽，安装蜂窝天线
2. 登录设备Web管理界面
3. 进入【Internet】→【Cellular】菜单，点击"编辑"按钮
4. 如需使用专网APN，填写APN账号密码（从运营商获取）
5. 如需使用双卡或SIM卡切换功能，在"工作模式"处选择相应选项
6. 点击"保存"按钮，等待设备自动拨号

<p align="center"><img src="./img/DyQS3nSv9PkPnk9T/1747634437501-ce633df6-05d5-43cf-a032-1ac9f6a50ba1-508332.webp" alt="蜂窝接口配置"></p>

<p align="center"><strong>图 3-2 设备蜂窝接口配置</strong></p>

**验证方法**：

1. 查看设备"网络"指示灯，确认绿色常亮（蜂窝已连接）
2. 进入【状态】→【链路监控】，查看蜂窝接口状态
3. 进入【状态】→【蜂窝信号】，查看信号强度

**常见问题**：

- 网络连接失败：检查SIM卡是否正确插入、APN参数是否正确
- 数据发送接收异常：检查信号强度和流量余额

### 场景3：Wi-Fi(STA)联网

**目标**：通过Wi-Fi（STA模式）接入现场已有的Wi-Fi网络。

**前提**：现场有可用的Wi-Fi网络，已知SSID和密码。

**预计用时**：约5分钟。

**操作步骤**：

1. 登录设备Web管理界面
2. 进入【Internet】→【Wi-Fi(STA)】菜单
3. 填写目标Wi-Fi网络的SSID和密码
4. 选择工作频段（2.4GHz或5GHz）
5. 点击"保存"按钮

### 场景4：配置IPSec VPN

**目标**：通过IPSec VPN隧道与远端设备建立安全连接。

**前提**：两端设备均支持IPSec VPN功能，已知对端设备的公网IP地址或域名。

**预计用时**：约10分钟。

**操作步骤**：

1. 登录设备Web管理界面
2. 进入【VPN】→【IPSec VPN】菜单，点击"添加"按钮
3. 配置以下参数：
   - **名称**：VPN隧道的本地标识
   - **IKE版本**：选择IKEv1或IKEv2
   - **预共享密钥**：两端设备配置相同的认证密钥
   - **Internet接口**：本端用于建立VPN的上行接口
   - **隧道模式**：选择隧道模式或传输模式
   - **本地子网**：需要通过VPN隧道通信的本端网段
   - **对端子网**：需要通过VPN隧道通信的对端网段
   - **对端地址**：对端设备的公网IP地址或域名
4. 配置IKE策略和IPSec策略（可使用默认配置）
5. 点击"保存"按钮

### 场景5：SD-WAN组网

**目标**：通过小星云管家平台快速创建SD-WAN网络，实现多分支互联。

**前提**：

- 所有设备均已获得"分支专业版"许可
- 设备均可联网并已配置完成
- 中心设备需要公网IP地址

**预计用时**：约15分钟。

**操作步骤**：

1. 登录小星云管家平台（https://star.inhandcloud.cn）
2. 进入【网络】→【SD-WAN】菜单，点击"添加"按钮
3. 填写网络名称和描述
4. 添加中心设备（Hub）：
   - 选择设备
   - 配置公网地址或映射地址
   - 勾选参与SD-WAN通信的子网
5. 添加分支设备（Spoke）：
   - 选择设备
   - 勾选参与SD-WAN通信的子网
   - 如有需要，配置静态路由
6. 点击"保存"按钮，等待网络创建完成

### 4.4 VPN

虚拟专用网络（VPN）是一种加密技术，用于在公共互联网上创建安全的、私人的网络连接。它允许用户通过互联网安全地访问私人网络资源，无论他们身在何处。VPN通过加密通信数据，确保通信的机密性和安全性，防止未经授权的访问。

#### 4.4.1 IPSec VPN

IPsec（Internet Protocol Security）VPN 是一种用于加强网络通信安全性的协议套件，旨在通过加密和认证来保护数据的传输。它广泛应用于建立安全的远程访问、站点到站点连接以及虚拟专用网络（VPN）。

ER815既可用作IPsec Server，也可以作为Client。

**入站/出站规则**

可通过"安全 > 防火墙 > 出站规则/入站规则"功能来实现基于接口的流量进出控制。

### 4.8 SD-WAN

企业分支机构之间，往往需要进行通信，以满足业务数据传输、视频会议等需求。传统的VPN配置比较复杂，故障排除可能具有挑战性。北京映翰通引入了SD-WAN网络功能，通过用户友好的界面，帮助用户在分支机构之间快速创建连接。

<p align="center"><img src="./img/DyQS3nSv9PkPnk9T/1751428258949-aba6daf1-98ca-4076-87ea-d50b33da1381-844128.webp" alt="SD-WAN应用场景"></p>

**前提条件**：

- 所有用于SD-WAN组网的设备都需要具有"分支专业版"许可
- 设备均可联网并已按规划配置完SD-WAN网络中需要通信的网段

#### 4.8.1 创建SD-WAN网络

在小星云管家平台的"网络"功能中，选择"SD-WAN"，点击"添加"，跳转到SD-WAN添加的页面。

<p align="center"><img src="./img/DyQS3nSv9PkPnk9T/1718179670412-e4977bfa-5919-4796-830c-8fa0db188a5c-152618.webp" alt="SD-WAN网络入口"></p>

<p align="center"><strong>图 4-64 SD-WAN网络入口</strong></p>

<p align="center"><img src="./img/DyQS3nSv9PkPnk9T/1742784193137-b6013d5c-e4fd-41f4-8a5f-806992b60ed9-501297.webp" alt="编辑SD-WAN网络"></p>

<p align="center"><strong>图 4-65 编辑SD-WAN网络</strong></p>

当前的SD-WAN网络支持中心辐射型拓扑，设备角色分为中心设备和分支设备。所有分支设备都通过中心设备创建隧道，分支设备之间的流量通过中心设备。

**中心设备要求**：

1. 中心设备需要公网IP地址来确保SD-WAN网络的运行。用户还可以通过IP映射来解决公网IP不足的问题。
2. 在具有公网IP地址的中心设备上行接口与分支设备的所有上行接口之间创建隧道。
3. 在防火墙规则中，中心设备的上游设备需要允许两个端口号，并将它们映射到ER系列路由器的上游接口。端口范围为1-65535。
4. 支持的设备型号：ER805、ER815、ER2000
5. 最多可以添加5个设备。

**分支设备要求**：

1. 分支设备对公共IP地址没有特定要求。
2. 可以添加多个分支设备，最终数量由中央设备的性能决定。
3. 支持的设备型号：ER605、ER805、ER815

### 案例1：连锁企业SD-WAN组网

**场景描述**：连锁企业有多个分支机构，需要实现分支与总部之间的安全互联，同时分支之间需要共享数据和资源。

**网络拓扑**：

<p align="center"><img src="./img/DyQS3nSv9PkPnk9T/1742788034294-160de567-114a-40bf-939e-cf016827676e-672361.webp" alt="分支SD-WAN场景"></p>

**设备角色**：

- 总部ER815作为SD-WAN中心设备（Hub），承担分支汇聚和转发功能
- 分支ER815作为SD-WAN分支设备（Spoke），接入各自本地网络

**配置步骤**：

1. 在总部部署ER815，配置公网IP地址
2. 在各分支部署ER815，配置蜂窝或有线联网
3. 登录小星云管家平台，创建SD-WAN网络
4. 添加总部设备作为中心设备，配置公网地址
5. 添加各分支设备作为分支设备
6. 配置各设备参与SD-WAN通信的子网
7. 保存配置，等待隧道自动建立

**验证方法**：

1. 在平台查看SD-WAN网络拓扑，确认所有隧道状态正常
2. 从分支1设备ping分支2设备，确认连通性
3. 从分支设备ping总部设备，确认连通性

### 案例2：IPSec VPN站点互联

**场景描述**：两个异地办公点需要通过安全隧道实现业务系统互访。

**网络拓扑**：

| 设备 | WAN口IP | 本地子网 | 对端子网 |
|------|---------|----------|----------|
| 总部ER2000 | 10.5.3.1 | 192.168.100.0/24 | 192.168.2.0/24 |
| 分支ER815 | 10.5.3.2 | 192.168.2.0/24 | 192.168.100.0/24 |

**设备角色**：

- ER2000作为IPSec VPN服务器端
- ER815作为IPSec VPN客户端

**配置步骤**：

1. **总部ER2000配置**：

## 附录 安全注意事项

1. 设备应在规定的温度和湿度范围内使用。
2. 请勿在易燃易爆环境中使用本设备。
3. 电源连接前请确认电压符合设备规格要求。

> **警告**：非专业人员请勿打开设备外壳，触电风险。

**安全使用指南**：

1. 请使用原装电源适配器，避免因电源适配器和设备不匹配造成设备损坏。
2. 安装设备时请勿将设备安装在强电磁干扰环境下，避免与高功率设备过于接近。安装完成后请确认设备是否易发生晃动，避免因设备跌落造成设备损坏。
3. 请确认设备使用环境满足规格书所要求的温度和湿度。
4. 请定期检查设备线缆，包括网线、电源适配器连接线，保持线材清洁，如有发生破损请及时更换。
5. 清洁设备时，避免使用化学试剂直接喷洒在设备表面，避免损害外壳或内部组件。需要使用柔性的布料清洁产品。
6. 请勿私自拆修或改装设备，这将会导致安全问题并且设备会失去保修资格。

## FAQ 常见问题解答

### 问题1：云平台是否免费？

映翰通一直致力于为中小连锁机构提供优质的网络服务，用户使用云平台服务时，需要逐台购买许可费用，以享受丰富的云端功能。

### 问题2：设备如何添加到云平台？

1. 首先通过https://star.inhandcloud.cn/网址注册小星云管家登录账号
2. 使用注册账号登录到云平台，在设备菜单下点击"添加"，按照提示填写设备的序列号和MAC地址，即可完成添加。设备首次添加时，默认赠送1年免费基础版许可，后续用户可按需续费。

### 问题3：设备是否可以不用云平台？

可以。用户可以在本地完成绝大多数配置功能，但批量的配置下发、升级、SD-WAN、Connector等功能，需要结合云平台使用。

### 问题4：ER815与普通路由器的区别？

1. ER815作为边缘路由器：支持多种联网方式（有线/蜂窝）、多千兆网口和WiFi6芯片设计，支持各类终端设备接入，同时设备支持SD-WAN、带外管理和云管理。
2. 普通路由器：通常依赖固定的宽带连接，如DSL、光纤，通过有线连接到网络。普通路由器无统一的管理平台和防火墙、SD-WAN等高级特性。

### 问题5：4G/5G无法联网如何处理？

1. 物理环境：首先检查SIM卡是否插到正确的卡槽位，蜂窝天线是否全部安装。
2. APN设置：确保APN的配置信息与运营商提供的一致。
3. 检查设备连通性：登录到设备本地界面，通过系统自带的ICMP工具，ping 8.8.8.8测试是否连通，若可连通，则检查设备（如电脑、手机）与路由器的连通性。
4. 判断SIM卡是否正常：取出SIM卡，安装在手机中查看是否可以正常联网。
5. 重启：尝试将路由器断电，等待几秒钟，然后重新连接电源，重试网络连接。
6. 恢复出厂：将路由器恢复出厂，然后重新尝试。
