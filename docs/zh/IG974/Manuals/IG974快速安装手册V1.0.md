## **1 产品介绍**

InGateway974 (简称IG974)是映翰通面向工业物联网领域推出的新一代工业级5G边缘网关；其拥有强大的边缘算力、丰富接口及Python开发环境，在物联网边缘节点进行数据优化、实时响应、敏捷连接和智能分析；同时支持多种工业协议和主流IoT云平台，轻松实现工业设备上云，帮助加快企业数字化进程，提升企业竞争力。

## **5 安装IG974**

### 5.1 DIN轨安装与拆卸

### 5.1.1 DIN轨安装

DIN轨的安装板附在IG974后面板上，如下图所示：

![1724658625382-37f2a1cc-83ce-404b-81c1-ed7b28be5c98.png](./img/8fVOGspyaWk0BL5f/1724658625382-37f2a1cc-83ce-404b-81c1-ed7b28be5c98-959511.png)

安装步骤如下  
1、选定设备的安装位置，确保有足够的空间；

2、将DIN卡轨座的上部卡在DIN轨上，在设备的下端向上稍微用力按箭头2所示转动设备，即可将DIN卡轨座卡在DIN轨上，确认设备可靠地安装到DIN轨上，如下图所示：

![1724658684694-614be72d-71ff-47c1-b180-4e8e0dd8cd59.png](./img/8fVOGspyaWk0BL5f/1724658684694-614be72d-71ff-47c1-b180-4e8e0dd8cd59-675932.png)

### 5.1.2 DIN轨拆卸

拆卸IG974的方法为：

1、如下图箭头1所示，向下压设备使设备下端有空隙脱离DIN轨。

2、将设备按箭头2的方向转动，并同时向外移动设备的下端，待下端脱离DIN轨后向上抬设备，即可从DIN轨上取下设备。

![1724660811574-6bcbc834-e5a7-44fd-a536-6949708b96d5.png](./img/8fVOGspyaWk0BL5f/1724660811574-6bcbc834-e5a7-44fd-a536-6949708b96d5-076849.png)

### 5.2 壁挂式安装及拆卸

### 5.2.1壁挂式安装

安装IG974的具体步骤如下：

1、选定设备的安装位置，确保有足够的空间。

2、用螺丝刀把壁挂安装板安装在设备的后面，如下图所示

![1724662948979-7a312280-1e3c-4748-804c-477abea34aeb.png](./img/8fVOGspyaWk0BL5f/1724662948979-7a312280-1e3c-4748-804c-477abea34aeb-445798.png)

3、取出螺钉（与壁挂安装板配套包装），用螺丝刀将螺钉固定在安装位置上，然后下拉设备使设备处于稳定状态，如下图所示

![1748243136869-604887ca-05d9-4df5-9150-bec1bb6f8534.png](./img/8fVOGspyaWk0BL5f/1748243136869-604887ca-05d9-4df5-9150-bec1bb6f8534-623829.png)        ![1748243136842-bd3f3fc6-11fd-4d64-990c-77bb0ef54bc8.png](./img/8fVOGspyaWk0BL5f/1748243136842-bd3f3fc6-11fd-4d64-990c-77bb0ef54bc8-364520.png)

### 5.2.1 壁挂式拆卸

拆卸IG974的方法为：用一只手扶住设备，另外一只手把设备上端起固定作用的螺钉卸掉，即可把设备从安装位置拆卸掉。

### 6.7 拨码开关

![1748245376659-084da446-a3d6-4935-8f40-ec909e2da373.png](./img/8fVOGspyaWk0BL5f/1748245376659-084da446-a3d6-4935-8f40-ec909e2da373-441134.png)

如上图所示，可通过3位的拨码开关分别实现RS-485接口的A/B的上/下拉电阻，和A、B端间电阻的使能选择；通过使能电阻可提高RS-485总线的抗干扰能力及解决因接口芯片兼容性导致的乱码问题。

| **标识** | **描述** | |
| --- | --- | --- |
| PU | ON:使能COM RS-485 A上拉电阻<br/>OFF:未使能COM RS-485 A上拉电阻（默认） | |
| PD | ON:使能COM RS-485 B下拉电阻<br/>OFF:未使能COM RS-485 B下拉电阻（默认） | |
| T | ON: 使能终端匹配电阻<br/>OFF: 禁用终端匹配电阻（默认） | |

备注：但使能电阻，会减少RS-485总线允许最大的接入设备数量。

### 6.8 恢复出厂按键

有一个复位按钮用于系统恢复出厂，采用硬件方式恢复出厂设备步骤如下：

- 步骤1：设备上电后10内长按RESET键；
- 步骤2：ERR灯常亮时，松开RESET键；
- 步骤3：ERR灯熄灭后，再次按住RESET键，ERR灯闪烁时松开RESET键；等待ERR灯熄灭，表明恢复出厂设置成功。
