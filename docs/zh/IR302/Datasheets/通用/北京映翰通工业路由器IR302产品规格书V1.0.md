# <span style="color: green;">1. 产品概述

IR302 是北京映翰通网络技术股份有限公司推出的紧凑型工业级蜂窝路由器，专为物联网（IoT）和工业自动化场景设计。该产品基于高性能工业级处理器，集成 4G LTE 蜂窝网络、以太网、Wi-Fi 及丰富的工业接口，可在严苛环境下提供稳定、安全的网络连接。

IR302 采用无风扇散热设计，支持宽温宽压工作，具备完善的链路备份、VPN 加密、防火墙及远程管理功能，满足电力、交通、环保、智能制造等行业对设备联网的严苛要求。

# <span style="color: green;">2. 核心特性

## <span style="color: green;">2.1 工业级可靠性
- **宽温设计**：工作温度覆盖 -35°C ~ +70°C（标准版）、-40°C ~ +75°C（扩展版）
- **宽压电源**：支持 9~36V DC 直流输入，兼容 12V DC 适配器，具备反接保护和浪涌防护
- **无风扇设计**：全金属外壳自然散热，无运动部件，MTBF 显著提升
- **电磁兼容性**：通过工业级 EMC 测试，适应强电磁干扰环境

## <span style="color: green;">2.2 全场景网络接入
- **4G LTE**：内置 Quectel EC25-AF(D) 工业级模块，支持全网通及海外频段
- **Wi-Fi**：支持 802.11 b/g/n，可同时作为 AP 和 STA 使用，满足现场无线覆盖与桥接需求
- **以太网**：1 个百兆 WAN/LAN 可切换网口 + 1 个百兆 LAN 口，支持双网口隔离与链路备份
- **链路备份**：蜂窝/有线/Wi-Fi 多链路智能切换，保障业务永续
- **双 SIM 冗余**：支持双 SIM 卡网络备份，ICCID 绑定、信号阈值自动切换

## <span style="color: green;">2.3 企业级安全与 VPN
- **VPN 协议**：IPSec（IKEv1/v2）、GRE、L2TP、PPTP、OpenVPN、WireGuard、ZeroTier
- **高级加密**：支持 AES256 / SHA512 / ECP521（DH21）等军工级加密算法
- **防火墙**：访问控制、端口映射（50 条）、虚拟 IP 映射、DMZ、MAC-IP 绑定（20 条）、NAT、内容过滤、SIP ALG
- **Wi-Fi 安全**：802.1X 认证、Portal 认证（captive portal）、AP 黑白名单 / MAC 过滤、Wi-Fi 用户隔离、Wi-Fi 密码加密存储
- **访问安全**：用户登录失败锁定（连续失败自动锁定账户/IP）、PAM 认证参数配置
- **安全启动**：SPI Flash 版本回退限制机制，防止降级至不兼容版本导致设备变砖

## <span style="color: green;">2.4 智能化运维管理
- **远程管理平台**：支持 Device Manager（DM）批量管理与 InConnect Service（ICS）云管平台
- **DTU 功能**：支持串口 RS232/RS485 及以太网 DTU 模式，实现透明数据传输
- **SNMP/Trap**：标准 SNMP v1/v2c/v3 及 Trap 告警，兼容主流网管系统
- **WebUI/CLI**：图形化中文界面 + 完整命令行（show/ping/telnet/traceroute/reboot/arping 等）
- **SSH 客户端**：设备可作为 SSH client 连接远端服务器
- **HTTP API**：提供 HTTP和HTTPS协议管理接口，支持远程配置与状态查询
- **网络诊断**：内置网络抓包（tcpdump）、导出系统运行状态、Wake-on-LAN 远程唤醒

# <span style="color: green;">4. 产品创新点

## <span style="color: green;">4.1 双 SIM 智能冗余与蜂窝精细化运维

- **ICCID 绑定 + 多维度切卡策略**：支持信号强度（RSRP）阈值切换、丢包率触发切换、最大拨号次数限制，实现真正智能化的双卡网络备份，告别简单轮询式切卡
- **蜂窝信号参数实时上云**：RSRP、RSRQ、SINR、PCI、BAND 等关键射频指标通过 DM 平台实时上报，远程即可掌握现场网络质量，无需派工程师到场
- **IMS 参数配置**：支持 VoLTE 专网 IMS 配置，满足运营商专网语音融合通信需求

## <span style="color: green;">4.2 电力行业端到端安全解决方案

- **国网加密硬件模块**：选配 -SEC 型号内置国密算法芯片，支持 IEC101/IEC104 电力规约，满足配电自动化终端（FTU/DTU）的国网准入要求
- **固件签名验证 + 版本回退限制**：系统固件自带数字签名，防止固件被篡改；升级前通过镜像头机型标识匹配 + 完整性/签名校验 + 版本兼容性（防降级）约束实现固件-硬件兼容性准入校验，不匹配即拒升级机制，配合 A/B 冗余分区与失败自动回滚机制，防止误刷异构固件导致设备变砖，杜绝现场"变砖"事故


<div style="page-break-after: always;"></div>


## <span style="color: green;">4.3 军工级 VPN 加密升级

- IPSec VPN 支持 **AES256 / SHA512 / ECP521（DH21）** 高级加密套件，达到等保 2.0 及电力/军工行业合规要求
- 为跨境数据通道和敏感业务提供量子计算时代的前期安全保障

## <span style="color: green;">4.4 免开箱智能运维工具集

- **WebUI 内置网络抓包（tcpdump）**：无需携带笔记本和抓包工具，直接在浏览器中捕获并下载网络报文，现场排障效率提升数倍
- **一键导出系统运行状态**：完整导出设备配置、日志、诊断信息，远程技术支持可精准定位问题
- **Wake-on-LAN + HTTP API**：支持远程唤醒局域网内下游设备，HTTP和HTTPS协议管理接口实现与上层运维平台的自动化对接
- **SSH 客户端**：设备可主动作为 SSH 客户端 outbound 连接远端服务器，突破 NAT 限制

## <span style="color: green;">4.5 DTU 断网数据续传

- **TCP Server 通道扩展**：虚拟串口功能最大支持 32 路并发通道，满足多设备同时接入场景

## <span style="color: green;">4.6 紧凑型高可靠工业设计

- 相比同系列 IR615，IR302 体积更紧凑，采用 **1x1 Wi-Fi 天线方案** 实现成本与性能的平衡，在 10 米范围内仍可提供 31 Mbps 稳定吞吐量，满足绝大多数工业巡检和 SCADA 数据交互需求

# <span style="color: green;">6. 网络连接能力

## <span style="color: green;">6.1 蜂窝网络（Cellular）
- **制式支持**：取决于所选购的蜂窝模组，涵盖 4G LTE Cat.4（下行 150 Mbps / 上行 50 Mbps）、4G LTE Cat.1、3G WCDMA、2G GSM/EDGE 等
- **频段**：各模组支持频段不同，详见[8.1 订购信息型号命名规则](#81-型号命名规则)
- **双 SIM 管理**：支持双 SIM 卡网络备份，ICCID 绑定，信号强度阈值（RSRP）自动切换，丢包率触发切卡，最大拨号次数限制，最小连接时间控制
- **SIM 功能**：支持运营商专网 APN、PIN 码锁定、流量统计
- **IMS 配置**：支持蜂窝拨号 IMS 参数配置
- **短信**：短信远程重启/拨号控制、自定义命令、访问黑白名单

## <span style="color: green;">6.2 Wi-Fi 无线局域网
- **标准**：IEEE 802.11 b/g/n，2.4 GHz 频段
- **天线配置**：1x1 单天线，理论峰值 72 Mbps
- **工作模式**：AP 模式（默认）、STA 站点模式、AP+STA 混合模式
- **接入控制**：AP 黑白名单 / MAC 过滤、Wi-Fi 用户隔离（客户端互隔离）
- **AP 容量**：最大支持 32 个 STA 同时接入
- **实际性能**（典型测试环境）：

|距离|协商速率|吞吐量（LAN→WLAN 典型值）|
|------|----------|---------------------------|
|1 m|72 Mbps|54 Mbps|
|5 m|72 Mbps|41 Mbps|
|10 m|72 Mbps|31 Mbps|
|30 m|72 Mbps|14 Mbps（视障碍物情况）|

- **多客户端接入能力**（吸盘胶棒天线，典型值）：

|接入数量|LAN→WLAN|WLAN→LAN|
|----------|----------|----------|
|1 台|~29.7 Mbps|~20.8 Mbps|
|5 台|~20.8 Mbps|~13.3 Mbps|
|10 台|~19.9 Mbps|~12.1 Mbps|
|15 台|~16.5 Mbps|~2.7 Mbps|

> **注**：Wi-Fi 性能受环境干扰、障碍物及终端能力影响，以上数据为实验室典型值，仅供参考。

## <span style="color: green;">6.3 有线以太网
- **接口速率**：10/100 Mbps 自适应
- **工作模式**：WAN/LAN 可软件切换
- **协议支持**：PPPoE、DHCP Client、静态 IP

## <span style="color: green;">6.4 链路备份与冗余
- **备份策略**：4G ↔ 有线 WAN ↔ Wi-Fi STA 智能切换
- **检测机制**：链路保活检测（ICMP）、快速切换
- **VRRP**：支持虚拟路由冗余协议，双机热备
- **负载均衡**：多 WAN 流量分担（可选策略）

## <span style="color: green;">7.3 防火墙与安全

- **访问控制**：基于 IP、端口、协议、时间段的 ACL 规则；支持登录失败锁定（连续失败自动锁定账户/IP）

- **内容过滤**：URL 关键字过滤、MAC 地址黑白名单
- **应用层网关**：SIP ALG（VoIP 音视频传输支持）
- **攻击防护**：防 Ping of Death、防 SYN Flood

## <span style="color: green;">7.4 工业通信与 DTU
