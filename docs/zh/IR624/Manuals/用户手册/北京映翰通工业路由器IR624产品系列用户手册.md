# 第1章 认识设备

## 1.1 概述

IR624 系列工业级蜂窝路由器，集成 4G/5G、Wi-Fi、虚拟专用网络（VPN）等核心技术，不仅能提供不间断网络访问能力与全面安全防护功能，还可通过内置的物联网行业智能软件服务，为各类场景构建安全可靠的业务数据链路，助力企业推进数字化与网络化转型。该系列产品特别适用于无人值守设备及站点的联网需求，其嵌入的看门狗与多层链路检测机制，能从硬件到协议层保障通信稳定无间断，同时深度结合 InHand DeviceLive 云平台，支持用户轻松实现云端统一集中管理，实时掌握设备运行状态，在大幅节省部署与运维成本的同时提升管理效率，最终为大规模物联网部署提供一站式高可靠解决方案。

# 第2章 安装与首次使用

## 2.1 安装前准备

### 环境要求

| 项目 | 要求 |
|------|------|
| 电源 | 使用原装电源适配器 |
| 温度/湿度 | 需满足规格书要求 |
| 安装位置 | 避免强电磁干扰环境，避免与高功率设备过于接近 |
| 工具 | SIM卡、蜂窝天线、Wi-Fi天线、网线 |

> **注意**：安装完成后需确认设备是否易发生晃动，避免因设备跌落造成设备损坏。

### 准备工作

1. 安装4G/5G、Wi-Fi天线并插入SIM卡。
2. 连接电源和网线，任意LAN口连接PC。
3. 将PC的IP地址设置为与工业路由器处于同一网段。

设备LAN口默认启用DHCP Server 功能，将PC设置为"自动获取IP地址"，PC即可通过IR624获取IP地址，需确认该地址是否处于192.168.2.0网段。若PC无法通过DHCP的方式自动获取IP地址，则需手动将PC的IP地址配置为192.168.2.2~192.168.2.254中的任一地址；默认网关为192.168.2.1，子网掩码为255.255.255.0，DNS设置为8.8.8.8或从运营商获取的DNS地址。

### 网络接入方式说明

IR624支持5G/4G、有线、Wi-Fi（STA）三种网络接入方式。

#### 有线接入网络

IR624支持蜂窝、有线、Wi-Fi三种网络接入方式。设备WAN/LAN1端口支持"WAN/LAN切换"功能，该端口默认处于LAN模式，如需通过有线接入网络，需要在【Internet>上行链路列表】中手动添加WAN链路，并选择合适的联网方式（该端口支持DHCP/静态IP/PPPoE三种有线接入模式）。完成设置后，用网线将IR624的WAN口与上行设备连接即可接入网络。

# 第3章 常用场景配置

## 场景1：蜂窝联网

**目标**：通过4G/5G蜂窝网络接入互联网。

**前提**：已插入SIM卡并安装天线，设备已上电。

**预计用时**：约5分钟。

**操作步骤**：
1. 插入SIM卡并安装天线。
2. 设备上电，等待系统自动拨号。
3. 在通常情况下，IR624会自动创建拨号连接并连接到Internet。
4. 当需要配置APN参数时，进入【Internet】→【蜂窝】菜单，点击"编辑"按钮进入APN参数配置界面。
5. 配置APN参数（APN参数需从运营商获取）。
6. 点击"保存"并等待连接建立。

**验证方法**：
1. 检查蜂窝状态灯，确认绿色常亮。
2. 在【状态】→【链路监控】中查看蜂窝链路状态。

**常见问题**：
- 网络连接失败：检查SIM卡是否正确插入、APN参数是否正确。
- 数据发送接收异常：检查信号强度和流量余额。
- 部分专网环境下，需手动关闭【Internet】菜单下的"链路探测"功能，避免因探测不通导致蜂窝不能正常工作。

## 场景2：有线接入网络

**目标**：通过有线方式接入互联网。

**前提**：已获取上行网络参数，设备已上电且PC已连接LAN口。

**预计用时**：约5分钟。

**操作步骤**：
1. 登录设备Web管理界面。
2. 进入【Internet】→【上行链路列表】，点击"添加"按钮。
3. 选择WAN接口类型，配置联网方式（DHCP/静态IP/PPPoE）。
4. 点击"保存"。
5. 用网线将IR624的WAN口与上行设备连接。

**验证方法**：
1. 在【状态】→【链路监控】中查看WAN链路状态。
2. 通过设备自带的Ping工具测试外网连通性。

**注意事项**：
- 添加WAN接口后，原LAN1接口角色将切换成WAN。
- 删除WAN接口后，原WAN接口角色将会切换回LAN1。
- 删除WAN接口后，与WAN接口相关的静态路由、入站/出站规则、端口转发、策略路由、流量整形相关配置都将被删除。

## 场景4：Wi-Fi接入点配置

**目标**：配置IR624作为AP提供无线网络接入。

**前提**：设备已上电，已登录Web管理界面。

**预计用时**：约5分钟。

**操作步骤**：
1. 进入【Wi-Fi】→【Wi-Fi列表】菜单。
2. 点击"添加/编辑"按钮，配置SSID名称、认证方式、密码等参数。
3. 点击"保存"。

**验证方法**：
1. 使用无线终端搜索配置的SSID。
2. 输入密码后确认可正常连接并访问网络。

**注意事项**：
- 设备默认有2.4G、5G两个主SSID，主SSID频段不支持修改且不可删除。
- 已添加的SSID频段不支持修改，信道将自动与所属主SSID的信道保持一致。
- 若用户在【Internet】菜单创建Wi-Fi(STA）接口，与Wi-Fi(STA）接口相同频段的SSID都无法启用直到Wi-Fi(STA）接口被删除。

## 4.2 Internet

可在"Internet"功能下配置每个上行接口的参数和操作模式。IR624支持有线、蜂窝、Wi-Fi三种接入网络模式。最多可以支持三个上行链路，包括 WAN、蜂窝和 Wi-Fi （STA）。WAN和Wi-Fi（STA）接口需要手动添加，并可根据需要删除；蜂窝上行链路不可删除。

### 4.2.1 有线连接

IR624支持三种有线连接方式接入网络：DHCP、静态 IP 和 PPPoE。可通过单击"编辑"按钮来修改连接方式。默认选中"DHCP"模式。

<p align="center"><img src="./img/UoaqO8Frv-kRG1ER/1747990788720-36f29d4b-2741-4831-9a2c-9545ba9fa1f0-865057.webp" alt="流量策略"></p>

### 4.2.5 上行链路设置

可通过"Internet>上行链路设置"功能配置链路检测设置，并配置不同上行接口之间的协作模式。

<p align="center"><img src="./img/UoaqO8Frv-kRG1ER/1747990789787-27fc65cd-868f-4798-bd12-cf5bfc80bde0-899991.webp" alt="上行链路设置"></p>

## 4.3 本地网络

可在【本地网络】功能中自定义本地子网，包括配置本地局域网的地址段、VLAN ID、DHCP服务等参数。完成配置后，需进一步在【接口管理】中将这些配置应用到设备LAN口，或在Wi-Fi设置中将其应用到目标SSID。这一系列操作旨在确保客户端设备能够按照规划的网络地址顺利连接到本地网络。

<p align="center"><img src="./img/UoaqO8Frv-kRG1ER/1747990790501-2c2d48a1-3e33-4de0-861d-a801a6a07be4-781899.webp" alt="本地网络编辑"></p>

<p align="center"><strong>图 4-27 本地网络编辑</strong></p>

- 名称：用于标识网络，用户可通过此名称选择在"Wi-Fi"和"接口管理"中应用的网络。
- 模式：选择当前子网是2层透明模式或3层IP模式，默认"IP模式"。
- VLAN：将局域网划分到不同虚拟逻辑的网络中。默认所有的接口和Wi-Fi使用default（VLAN1）网络。
- IP地址/掩码：通过LAN口或Wi-Fi访问路由器的网关地址，默认"192.168.2.1"。

- 已添加的本地网络模式不支持修改。
- VLAN Only模式用于二层透传，无需配置IP地址/掩码、DHCP Server。

## 4.5 VPN

- 名称：路由器创建的IPSec VPN名称标识，主要用于本地管理。
- IKE版本：设置IKE协议使用的版本号，支持IKEv1和IKEv2。

- Internet接口：本地建立IPSec VPN使用的上行接口。
- 隧道模式：设置IPSec对IP报文的封装模式，支持隧道模式和传输模式。
- 对端地址：与IR624建立隧道的对端通信地址。

说明：

IR624在用IPSec VPN建立连接时，默认以使用公网地址的设备作为服务器，IPSec服务器侧的对端地址需要配置为 0.0.0.0，IPSec客户端侧的对端地址填写IPSec服务器侧的公网地址。

- 本地子网：填写需要通过IR624 IPSec VPN隧道通信的网段地址。
- 对端子网：填写隧道另一端需要通过IPSec VPN隧道通信的网段地址。
- IKE策略：支持设置IKE协议的配置。
  - 加密方式：设置IKE使用的加密算法。取值范围：DES、3DES、AES128、AES192、AES256，默认"AES128"。
  - 认证方式：设置IKE使用的认证算法。取值范围：MD5、SHA1、SHA2-256、SHA2-384、SHA2-512，默认"SHA1"。
  - DH组：设置IKE阶段密钥协商使用的DH交换参数。取值范围：1、2、5、14、15、16、19、20。
  - 超时时间：设置IKE SA存活时间，默认86400秒。
- IPSec 策略：支持设置IPSec参数配置。
  - 安全协议：设置ESP协议使用的安全协议。取值范围：DES、3DES、AES128、AES192、AES256，默认使用"AES128"。
  - 加密方式：设置ESP协议的加密算法。取值范围：MD5、SHA1、SHA2-256、SHA2-384、SHA2-512,默认"SHA1"。
  - PFS组：IPSec使用安全策略发起一个协商时，在阶段2的协商中进行一次附加的密钥交换以提高通讯的安全性参数。取值范围：1、2、5、14、15、16、19、20。
  - 超时时间：设置IPSec SA老化时间，默认86400秒。

## 4.6 安全

在【安全】菜单下，可根据需要设置防火墙、策略路由、流量整形相关的高级功能。

### 4.6.1 防火墙

防火墙当前包括：入站规则、出站规则、端口转发、MAC地址过滤、域名过滤等功能，可根据具体的业务需求进行配置。

**入站/出站规则**

可通过"安全 > 防火墙 > 出站规则/入站规则"功能来实现基于接口的流量进出控制。例如，当受到来自特定源IP地址的大量攻击行为时，可以使用防火墙入站规则来限制来自该IP地址的流量。此外，还可通过防火墙出站规则来限制某些用户访问外部网络的权限。入站规则和出站规则在可配置的内容上相同，唯一的区别是默认规则。以下是添加入站规则的示例：

<p align="center"><img src="./img/UoaqO8Frv-kRG1ER/1747990794057-555e6e56-df23-4e64-8be0-d1f2760d674c-199950.webp" alt="防火墙入站规则"></p>

<p align="center"><strong>图 4-36 防火墙入站规则</strong></p>

- 名称：设置入站规则的名称，用于本地标识。
- 状态：规则的功能开关。
- 接口：对于出站规则，指流量流出路由器的上行接口。对于入站规则，指流量进入路由器的上行接口。
- 协议：匹配流量的协议类型，支持Any、TCP、UDP、ICMP、自定义。
- 源：匹配流量的源地址，支持自定义，默认Any。
- 目的：匹配流量的目的地址，支持自定义，默认Any。
- 动作：入站规则/出站规则的对匹配流量的动作，支持允许和拒绝。

**端口转发**

端口转发也被称为端口映射或端口重定向，用于将网络数据包从一个网络端口（或地址）重定向到另一个网络端口或地址。可在"安全>防火墙>端口转发"配置端口转发规则，当外部流量访问路由器的特定端口时，设备会将数据转发至内部客户端的相应端口上，实现从外部访问路由器内部的服务。

**例如：** 当需要从外部访问内部192.168.2.10客户端的1024端口服务时，将此客户端的端口映射到WAN1接口下的1024端口，外部网络只需在浏览器输入 `https://WAN1地址:1024` 即可访问目标设备数据，其中WAN1地址为WAN1接口的IP地址。

- 名称：端口转发规则的名称，用于本地标识。
- 状态：端口转发规则的功能开关。
- 接口：对内部客户端提供映射功能的上行接口，上行接口需要公网地址支持。
- 协议：端口映射流量的协议类型，支持TCP、UDP、TCP&UDP。
- 公网端口：对内部客户端提供映射功能的上行接口的端口号，需要与本地端口输入范围保持一致。
- 本地地址：外部网络需要访问的位于路由器下的目标设备地址。

**网络地址转换**

网络地址转换是一种将IP数据包在通过路由器时重新包内源IP地址或目的IP地址的技术，可以将私网IP地址转换为合法IP地址。可在"安全>防火墙>网络地址转换"中配置源IP地址转换或目的IP地址转换规则。

  - DNAT：目的IP地址转换，可将IP数据包中目的IP地址字段的值转换为设定值。

- 转换后地址：根据所选择的地址转换类型，设置需要转换的值。

### 4.9.3 远程访问控制

<p align="center"><img src="./img/UoaqO8Frv-kRG1ER/1747990800930-a306fbf4-901f-4336-a61c-5ee34e3ba7ea-121495.webp" alt="远程访问控制"></p>

- SSH：启用后，可以在远程工具（如CRT），输入设备上行接口的公网地址&端口、用户名及密码，实现远程登录路由器后台。

### 4.9.9 计划重启

计划重启是一种网络设备管理的策略，允许管理员在特定时间或条件下自动重启设备，以确保设备的正常运行和性能。

# 附录 安全注意事项

1. 请使用原装电源适配器，避免因电源适配器和设备不匹配造成设备损坏。
2. 安装设备时请勿将设备安装在强电磁干扰环境下，避免与高功率设备过于接近。安装完成后请确认设备是否易发生晃动，避免因设备跌落造成设备损坏。
3. 请确认设备使用环境满足规格书所要求的温度和湿度。
4. 请定期检查设备线缆，包括网线，电源适配器连接线，保持线材清洁，如有发生破损请及时更换。
5. 清洁设备时，避免使用化学试剂直接喷洒在设备表面，避免损害外壳或内部组件。需要使用柔性的布料清洁产品。
6. 请勿私自拆修或改装设备，这将会导致安全问题并且设备会失去保修资格。
