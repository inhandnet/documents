# SD-WAN配置示例

## 背景

一家连锁品牌店希望通过SD-WAN网络实现分支机构与总部的数据互通，并且需要确保与现有内部网段`172.16.10.0/24`能无缝通信。

![1741855698756-a2adc00f-ab9c-4483-87e2-8738c1983a9f.png](./img/aDdeCDpkd_r8Ups_/1741855698756-a2adc00f-ab9c-4483-87e2-8738c1983a9f-913227.png)

**【网络现状】**

门店1（分支）：ER605.  蜂窝/WAN：10.1.1.1，子网：192.168.1.1/24

门店2（分支）：ER605.  蜂窝/WAN：10.1.2.1，子网：192.168.2.1/24

总部（中心）：ER2000-Plus.  WAN口公网IP：10.1.1.1，子网：192.168.4.1/24

客户内网：172.16.10.0/24

**【预期效果】**

门店1（192.168.1.1/24）和门店2（192.168.2.1/24）的子网客户访问到总部网络（192.168.4.1/24）。整个SD-WAN网络能与客户现有的内网（172.16.10.0/24）实现互通。

## 前置条件

- 登录账号角色为 **系统管理员**。
- 这三台设备已经应用 \*\*小星云管家分支专业版 \*\*许可证，应用步骤请参考：[应用/取消许可](https://inhandnetworks.yuque.com/unup4w/ntla5f/lb3smox06sbm51el#133)。
- 设备在线。

## 步骤一：创建SD-WAN网络并配置网络基础信息

1. 使用系统管理员账号登录到[小星云管家](https://star.inhandcloud.cn/)；
2. 在左侧菜单栏中点击 **网络 >> SD-WAN网络；**
3. 点击 **添加**；
4. 输入 **网络名称**，如：Test；
5. 选择 **网络连接类型**，如：交叉互联；
6. **所属组织** 与 **强制转发** 可根据实际情况配置。

![1741847285321-f8d55f56-6e57-450d-8a9f-e86f6fe824c3.png](./img/aDdeCDpkd_r8Ups_/1741847285321-f8d55f56-6e57-450d-8a9f-e86f6fe824c3-265983.png)

![1741847320791-6818cd87-036c-4e4e-8fa6-3ca1c630cf6f.png](./img/aDdeCDpkd_r8Ups_/1741847320791-6818cd87-036c-4e4e-8fa6-3ca1c630cf6f-448928.png)

## 步骤二：添加中心设备

1. 点击中心设备下的 **添加** 按钮；
2. 下拉选择 中心设备；
3. 配置中心设备的以下参数：
   1. **隧道端口**：500/4500；
   2. 添加WAN口的**公网IP映射**：10.1.1.1，并保存；
   3. 选择**子网**：192.168.4.1/24。
4. 点击 **确定**。![1741847846273-c9822a47-d2cb-41f9-bfcd-e8ce409b63b6.png](./img/aDdeCDpkd_r8Ups_/1741847846273-c9822a47-d2cb-41f9-bfcd-e8ce409b63b6-166167.png)

:::info
\*\*提示：\*\*如果页面展示的子网，没有您需要配置在SD-WAN的，可以点击 **编辑** 按钮，到设备的 \*\*本地网络列表 \*\*进行添加。

:::

## 步骤三：宣告外部路由

:::info
\*\*提示：\*\*如果您不需要与其他网络互通，可忽略此步骤。

:::

1. 编辑中心设备的 **子网，** 进入中心设备的配置编辑页面**；**
2. 进入 **服务 >> 静态路由** 页面；
3. 点击 **添加**，
4. 配置客户内网。
5. 点击 **保存**，并 **提交配置**。
6. 在编辑中心设备弹窗中，**勾选配置的路由**，并 **保存**。

![1741851172289-f9530d57-c26d-4615-a1d0-c3b090159f2e.png](./img/aDdeCDpkd_r8Ups_/1741851172289-f9530d57-c26d-4615-a1d0-c3b090159f2e-919718.png)

## 步骤四：添加分支设备

1. 点击分支设备下的 **添加** 按钮；
2. 下拉选择 分支设备，如：门店1、门店2；
3. 配置分支设备的以下参数：
   1. 可配置优选中心为总部设备，此示例未配置。
   2. 选择**子网**：192.168.1.1/24。
4. 点击 **确定**。
5. 按以上步骤添加其他所需的分支设备。此示例还添加了一个子网为192.168.2.1/24的分支设备。
6. 分支设备添加完成后，点击左下方的 **保存** 按钮，一个SD-WAN网络即可创建成功。

![1741847948232-0d04617f-fc3b-4381-b5e1-b98ba4b118e5.png](./img/aDdeCDpkd_r8Ups_/1741847948232-0d04617f-fc3b-4381-b5e1-b98ba4b118e5-524003.png)

:::info
\*\*提示：\*\*如果页面展示的子网，没有您需要配置在SD-WAN的，可以点击 **编辑** 按钮，到设备的 \*\*本地网络列表 \*\*进行添加。

配置完SD-WAN网络后，系统将会自动将配置推送到设备。

您可以通过SD-WAN拓扑图，查看分支设备与中心设备的隧道建立情况。

## 常见问题

### 设备添加到SD-WAN网络后，提示“当前许可不支持此功能”

![1741855471774-8328d6cd-5711-4992-9de3-b5da5d262b17.png](./img/aDdeCDpkd_r8Ups_/1741855471774-8328d6cd-5711-4992-9de3-b5da5d262b17-031977.png)

当设备无许可或者许可类型非小星云管家分支专业版事，会出现此异常提示。您需要到[映翰通云服务](https://portal.inhandcloud.cn/license/devices)的 **订阅** 菜单，为设备应用 **小星云管家分支专业版** 许可证。

许可证应用步骤请参考：[应用/取消许可](https://inhandnetworks.yuque.com/unup4w/ntla5f/lb3smox06sbm51el#133)。

![1741855372334-3b2cb087-b094-4ad8-8044-ab36056236cb.png](./img/aDdeCDpkd_r8Ups_/1741855372334-3b2cb087-b094-4ad8-8044-ab36056236cb-980691.png)

### 无法将设备添加到SD-WAN网络中

一般是由于您向选择的设备暂不支持SD-WAN功能。目前支持SD-WAN功能的设备型号：**ER2000、ER805、ER605。**

### 如何将所有分支业务流量通过优选中心设备访问？

您需要：

1. 开启SD-WAN网络的\*\* 强制转发\*\* 功能；
2. **编辑** 分支设备，为其选择 **优选中心**；
3. \*\*保存 \*\*以上配置。
