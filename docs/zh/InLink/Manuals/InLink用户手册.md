# InLink用户手册

## 产品简介

物联网连接服务InLink，是面向企业用户提供的一站式物联网连接服务，包括物联网卡及卡管理平台，旨在为用户提供便捷、快速和安全的SIM卡管理服务。

客户采购物联网卡后，可在[InLink平台](https://link.inhandcloud.cn/)实现卡的一站式管理。

InLink主要功能包括：

- SIM卡监控：监控SIM卡的状态、流量使用情况、服务到期情况等；
- SIM卡批量管理：批量修改SIM卡状态、修改白名单等；
- 在线充值续费：支持流量包充值、套餐续费、服务开通等；
- 自动化规则：自动告警SIM卡的状态切换及流量使用情况。

### 名词解释

### 1.物联网卡与人联网卡

- **【物联网卡】**

物联网卡是运营商为物联网服务企业提供的用于智能终端设备联网的SIM卡，广泛用于共享单车、移动支付、智能城市、自动售卖机等领域。物联网卡不能用于手机等人联网设备。

- **【人联网卡】**

人联网卡，通常就是我们日常所说的手机SIM卡，主要用于手机、平板等个人终端设备的网络连接。

- **【物联网卡与人联网卡的区别】**

| **特性** | **物联网卡** | **人联网卡** |
| --- | --- | --- |
| **用途** |  设备间通信（如智能家居、工业设备）   |  常用于个人终端设备（如手机、平板等），也可以用于物联网设备 |
| **服务类型** |  仅数据服务   |  数据、语音、短信  （目前映翰通提供的人联网卡仅用于数据服务） |
| **连接稳定性** |  高可靠性、长时间稳定   |  高速、低延迟   |
| **应用场景** |  物联网设备（传感器、智能硬件等）   |  日常通信设备   |
| **访问限制** | 只能访问特定的地址 | 访问一般不受限制 |
| **使用限制** | 参考【物联网卡使用限制】 | + 必须实名认证 |

:::danger
**【物联网卡使用限制】**

- **定向流量：** 物联网定向流量是在使用物联网卡接入网络时，只允许用户终端访问客户预先设置的业务平台或应用系统，不允许访问任何其他网络和服务。
- **无法访问人联网设备或应用**：物联网卡默认是企业实名制的，只能用于物联网设备，不能用于人联网设备（包括但不限于手机等）和访问人联网应用，人联网应用如：
  - 社交类：如微信、QQ、微博等。
  - 购物类：如淘宝、京东等。
  - 视频类：如抖音、快手、爱奇艺等。
  - 游戏类：如QQ游戏等。
- **机卡绑定：** 物联网卡统一设置了“机卡绑定”功能，当卡首次插入设备A激活成功时，即和设备A自动绑定。如果物联网卡被换到了设备B，将触发机卡分离停机，需要机卡解绑/重绑，才能在设备B上正常使用。
- **达量断网：**
  - 单卡达量断网：物联网卡统一设置了“达量断网”功能，单卡当月已用流量超出套餐额度则自动断网，次月自动恢复。单卡当月剩余流量不累计到下个月。
  - 流量池达量断网：流量池内单卡已用流量超过套餐额度可正常使用，流量池内所有卡当月已用流量总和超出流量池内SIM卡的套餐额度总和则自动断网。流量池当月剩余流量（如果有的话）不累计到下个月。
- **区域使用限制**：物联网实体卡仅限中国内地（新疆、西藏、广西、云南边境等高风险区域除外，上述区域需要客户到当地运营商自行购买）使用，不能在海外区域使用。
- **不提供语音和短信功能**：物联网卡只提供流量功能，不提供语音和短信功能。
- **不支持更换运营商**：下单后不支持更换运营商。
- **不支持指定号段**
- 不能放入手机使用，可能被运营商判定为异常导致停机。

:::

### 2.单网卡和多网卡

- **【单网卡】**

内置一个运营商的芯片，只能使用这个运营商的服务。手机卡就是单网卡。

- **【多网卡】**

内置多个运营商的芯片，可以切换使用中国移动、中国联通、中国电信这些运营商的服务。具体可使用的运营商可在购卡时选择。

- **【单网卡与多网卡的区别】**

| **特性** | **单网卡** | **多网卡** |
| --- | --- | --- |
| **网络连接数** |  仅通过一张网卡连接   |  通过两张或更多网卡连接多个网络   |
| **冗余与容错** |  无冗余，一旦失效无法连接   |  提供冗余和容错能力，网络失败时可切换   |
| **带宽与性能** | 满足基本的网络连接和数据传输需求 | 提供更稳定、更快速的网络连接，满足高并发、大数据量传输等需求 |
| **管理与配置** | 无需特殊配置 | + 使用多网卡的设备需适配多网卡SDK<br/>+ 可远程配置切卡策略 |
| **成本** |  较低，配置简单   |  较高，需要更多配置和管理   |
| **适用场景** |  对网络性能要求不高的应用/场景，如手机、普通路由器等   |  需要高可靠性和高带宽的应用/场景，如企业级路由器   |

### 材质与规格

- 消费级卡：普通塑料等材质，适用于普通消费使用场景。
- 工业级卡：注塑、陶瓷等材质，面对需要特殊环境中使用的场景，工业级卡有耐高温、防潮湿、抗震动、抗腐蚀、抗电磁干扰、使用寿命长等优势。

### 2.卡的规格

物联网卡的尺寸如下，单位为毫米：

- 大卡：25*15、Micro卡：15*12、Nano卡：12.3*8.8、贴片卡：5*6、贴片卡：2*2。
- 三切卡：指同时支持大卡、Micro卡和Nano卡三种尺寸的物联网卡。即客户不需自行裁剪，可以用三种不同的方式把物联网卡从卡槽中掰下来。
- 两切卡：指同时支持大卡、Micro卡两种尺寸的物联网卡。

### 3.卡号

- ICCID：物联网卡的卡号，用于InLink平台对物联网卡的管理，比如查询卡信息、充值加油包、续费套餐等。
  - 移动卡：ICCID为20位字符，与实体卡上的号码一致；
  - 联通卡：ICCID为20位数字。**实体物联网上的卡号最后一位为字母，卡添加到InLink平台上时，系统将自动转换成数字，因此在InLink平台管理物联网卡时，请使用平台上的ICCID。**
  - 电信卡：ICCID为19位字符，与实体卡上的号码一致。
- 多网卡ID：多网卡的卡号，为9位字符。在InLink平台对物联网卡的管理，比如查询卡信息、充值加油包、续费套餐等，都需通过多网卡ID进行。

## 快速入门

### 采购SIM卡

请联系销售咨询采购。[联系我们](https://www.inhand.com.cn/company/contact-us/)  

购卡注意事项：

1. 购卡前请先在[InLink平台（link.inhandcloud.cn）](https://link.inhandcloud.cn/)平台注册账号，并在订单里备注账号名称。
2. 购卡时请明确需求，包括流量套餐、SIM卡数量、定向地址清单、是否组流量池
3. **定向地址说明：**

所有SIM卡默认包含以下定向地址：

1. any://*.amazonaws.com.cn
2. any://*.inhandcloud.cn
3. <https://www.baidu.com>
4. 54.223.100.252
5. 54.222.145.115
6. 116.62.113.170
7. any://*.ntp.org（用于设备同步时间）  

以上地址可实现对InHand 以下平台的定向访问：Device Manager、InConnect、iSCADA、白鹰能源管家平台、小星云管家、DeviceLive平台。

如需访问其他地址，请在购新卡时向客户经理告知明确的需求。每张SIM卡最大可绑定10个定向IP或域名，如需变更已有卡的定向地址，请在InLink平台提交定向申请，定向申请需提供给运营商修改，一般情况预计3-5个工作日完成，请合理规划要访问的地址，避免更改地址影响业务测试或上线。

所有卡不单独售卖，需搭配硬件一同出售。一经出售，不能退款，请慎重下单。

**流量池说明**：SIM卡默认不组流量池，如需要组流量池，请在购卡告知客户经理。

### 管理SIM卡

采购发货后，您可以使用采购时提供的平台账号登录InLink平台，便可看到SIM卡。

**SIM卡激活：**

- 测试期到期强制激活

测试期默认3个月，超过3个月卡自动激活，进入计费。

- 测试期流量用完强制激活

测试期流量默认2MB，2MB用完卡自动激活，进入计费。  

**定向地址变更：**

如需变更定向地址，请在InLink提交定向申请。定向申请需提供给运营商修改，一般情况预计3-5个工作日完成，请合理规划要访问的地址，避免更改地址影响业务测试或上线。

### 停机

![1735892899099-aa220f25-f649-4381-942d-0fcf004ae061.png](./img/I1yPoNd-hV36sN7O/1735892899099-aa220f25-f649-4381-942d-0fcf004ae061-858724.png)

SIM卡存在两种停机方式：

- 系统自动停机：当SIM卡的周期流量用尽或欠费，系统将对SIM卡进行停机。
- 主动停机：用户选择单个/多个SIM卡，点击“批量编辑>>停机”，二次确认后，即可对选中的SIM卡进行停机。

> **注意：** 由于运营商原因，可能导主动致停机失败，如需帮助，可提交工单联系我们。

### 复机

![1735893009268-2850a785-26ef-4e94-ba8b-ee66417e51ea.png](./img/I1yPoNd-hV36sN7O/1735893009268-2850a785-26ef-4e94-ba8b-ee66417e51ea-109762.png)

SIM卡存在多种复机方式：

- 系统自动复机：
  - 充值流量包复机：月卡因为超套停机后，可以充值流量包恢复当月的服务；充值后，大部分设备无需重启即可正常联网。如果发现设备无法联网，重启即可。
  - 续费套餐复机：服务到期的SIM卡在套餐续费成功后，系统将自动复机；充值后，大部分设备无需重启即可正常联网。如果发现设备无法联网，重启即可。
  - 系统次月自动复机：月卡超套停机后，如果服务周期未结束，将在次月1日自动复机。
- 主动复机：选择单个/多个SIM卡，点击“批量编辑>>复机”，二次确认后，即可对选中的SIM卡进行复机。

> **注意：** 由于运营商原因，可能导致复机失败，如需帮助，可提交工单联系我们。

### 策略与规则

### 多网卡策略

为了保障多网卡的信号质量，可以基于当前卡的信号、Ping检测、Ping延时等情况，配置自动切卡的策略。操作步骤：

1. 进入“**策略与规则 >> 多网卡策略**”;
2. 点击“新增”；
3. 配置所需的字段信息并提交；
4. 如需应用到多网卡，可到多网卡列表进行“**批量操作 >> 切换策略**”。

策略字段含义如下：

| **字段** | **说明** |
| --- | --- |
| 策略名称 | 自定义策略名称。 |
| 自动切网 | 是否开启自动切换。若关闭则设备SDK端断网也不执行切换操作，连同下发检测策略也不会执行。 |
| 信号检测 | 对模组的RSRP信号进行检测，若低于设置的值进行切换。<br/>大部分场景下，-44dBm~-80dBm连接均为良好，对网络带宽几乎没有影响；当-80dBm~-109dBm时，网络带宽随着信号的减少而呈现线性下降，通过实测-109dBm时，网络带宽约为1Mbps，-102dBm时，网络带宽约为8Mbps，-80dBm时约为50Mbps。<br/>当设备对设备带宽有要求时，如视频监控场景，可选择对信号阈值进行设置。但设置前需要了解当前三个运营商的信号值，避免频繁切换。若其他场景对带宽无准确要求时，建议不开启信号检测，即可在信号极差时也能保证连接。 |
| PING检测 | 对需要监控的单个或多个地址URL或IP设置，SDK优先监控设置的地址。当设置的地址多次不通时，且检测内部的地址。全部地址多次不通进行切换。 |
| PING延时 | 优先对设置的URL或IP进行延时检测，延时多次大于设置的值时，进行切换。推荐不要设置低于-300ms，否则可能出现频繁切换的情况。 |
| 网络制式检测 | 设置网络制式黑名单，比如4G设备设置2G/3G时，设备回落进行切换。不过当前大部分模组及运营商已经不支持2G/3G的频段，所以4G设备无需设置此检测。<br/>主要为提供5G设备回落4G时使用，当前5G基站分布不均，如移动有5G的地点电信没有5G，所以此功能针对部分路线明确的可移动设备。 |
| 运营商优先级 | 设置最优和最低优先级，则按对应运营商顺序进行切换。否则按照默认顺序电信-移动-联通进行切换。 |

:::danger
【说明】

- 切换策略与切换运营商操作不冲突。

:::

## 常见问题

### 物联网卡的使用问题

**1. 卡插入设备后为什么无法上网？**

如果您的卡插入设备后，无法正常使用，请检查以下内容：

1. 卡是否正常安装：插卡后，设备的蜂窝信号灯是否正常；
2. 卡的状态是否正常：在平台查看卡是否已停机，或将SIM卡插入其他设备看能否正常使用；详细说明请参考"单网卡状态说明"文档。
3. 检查设备是否正常：插入其他正常工作的SIM卡后，查看设备是否可用；
4. 定向地址是否正确：如果卡未配置需要访问的地址，可在平台提交定向申请。

如以上都核实无问题，请联系客户经理。

**2. “未激活”状态的卡怎么激活？**

“未激活”的卡没有套餐，当在“**业务办理与计费 >> 充值续费 >>开通服务**”为卡购买套餐后，系统将自动激活卡。

**3. 卡的测试期是什么？什么时候结束？**

卡在测试期是可以正常使用的。

卡的测试期为3个月，期间共有2MB的流量供您测试使用。测试期从向映翰通下单之日起算。

当测试期的流量超过2MB或测试期结束后，卡将被自动激活，开始正式计费。  

**4. 卡的状态有哪些？**

详细说明请参考[单网卡状态说明](about:blank)。  

**5.** 卡换绑设备后，卡为什么停机了？**

工信部对物联网卡的使用有严格的限制，物联网设备的IMEI需要与物联网卡的IMSI一一对应，严禁违规使用或二次销售物联网卡。因此，卡首次插入设备后，就与设备机卡绑定，原则上不允许解绑。

如果由于设备损坏或其他原因确实需要解绑，请联系客户经理。  

**6.** 怎么更改卡的白名单地址？**

所有SIM卡默认包含以下6个定向地址：

- any://*.amazonaws.com.cn
- any://*.inhandcloud.cn
- <https://www.baidu.com>
- 54.223.100.252
- 54.222.145.115
- 116.62.113.170

如果你需要修改卡的白名单地址，可在InLink平台的“**业务办理与计费 >> 业务办理**”发起定向申请，详见业务办理。  

**7. SIM卡列表的信息是实时更新的吗？如何查看最新的卡数据？**

由于运营商侧限制，SIM卡列表不支持实时更新。

如果您需要查看卡的实时数据，可在卡详情手动刷新。

### 流量池相关问题

**1.** 什么是流量池？**

- 同一套餐规格的月结卡可以组成一个流量池，对池内的所有卡进行统一计费。
- 池内卡流量共享：一张卡的流量用尽后，可以使用另一张卡的流量。流量池的总流量用尽后，池内的卡才会因为流量用尽而停机。  

**2.** 怎么为流量池扩池？**

您可以通过以下任一方式扩池：

- **单卡充值加油包：** 为流量池中的卡充值加油包后，流量池将被重新激活使用。
- **向池内添加卡：** 可以向流量池中添加卡进行扩池。

如需为卡组建流量池，请联系客户经理。  

**3. 有没有流量池流量不足时的通知？**

您可以到“**策略与规则 >> 自动化规则**”配置流量池流量预警，支持邮件和短信通知。

### 充值续费相关问题

**1. 卡的套餐到期或流量用尽怎么办？**

- 套餐到期：可在平台套餐续费；
- 流量用尽：月卡当月流量用尽后，可充值加油包重新激活使用。

具体操作步骤，详见"充值续费"章节。  

**2. 充值后，卡的套餐什么时候生效？**
