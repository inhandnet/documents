# 第1章 产品简介

## 1.1 产品概述与定位

InLink是面向企业用户提供的一站式物联网连接服务平台，涵盖物联网卡及卡管理平台，旨在为用户提供便捷、快速和安全的SIM卡管理服务。用户采购物联网卡后，可在[InLink平台](https://link.inhandcloud.cn/)实现卡的一站式管理，包括SIM卡状态监控、批量管理、在线充值续费及自动化告警等功能。

## 1.2 核心功能与应用场景

InLink平台主要功能包括：

- **SIM卡监控**：监控SIM卡的状态、流量使用情况、服务到期情况等；
- **SIM卡批量管理**：批量修改SIM卡状态、修改白名单等；
- **在线充值续费**：支持流量包充值、套餐续费、服务开通等；
- **自动化规则**：自动告警SIM卡的状态切换及流量使用情况。

该平台适用于需要集中管理大量物联网SIM卡的企业用户，如智能硬件制造商、工业设备运营商、智慧城市解决方案提供商等。

## 1.3 名词解释

| 术语 | 通俗解释 | 适用场景 |
|------|----------|----------|
| 物联网卡 | 运营商为物联网服务企业提供的用于智能终端设备联网的SIM卡，不能用于手机等人联网设备 | 设备需要接入移动网络进行数据传输 |
| 人联网卡 | 日常所说的手机SIM卡，主要用于手机、平板等个人终端设备的网络连接 | 个人终端设备联网 |
| 单网卡 | 内置一个运营商芯片，只能使用该运营商服务的SIM卡 | 固定运营商网络场景 |
| 多网卡 | 内置多个运营商芯片，可切换使用中国移动、中国联通、中国电信服务的SIM卡 | 需要网络冗余和高可靠性的场景 |
| 定向服务 | 限制物联网卡仅能访问预先设置的白名单地址，保障使用安全 | 对访问安全性有要求的业务 |
| ICCID | SIM卡的卡号，用于InLink平台对物联网卡的管理 | 查询卡信息、充值续费等操作 |
| 流量池 | 同一套餐规格的卡组建的共享流量池，池内卡流量共享 | 多张卡共用流量配额 |
| 机卡绑定 | SIM卡首次插入设备激活后自动与该设备绑定，换设备会触发停机 | 防止违规转售或挪作他用 |
| 达量断网 | 当月已用流量超出套餐额度后自动断网，次月自动恢复 | 流量管控 |

### 1.3.1 物联网卡与人联网卡的区别

| 特性 | 物联网卡 | 人联网卡 |
|------|----------|----------|
| 用途 | 设备间通信（如智能家居、工业设备） | 常用于个人终端设备（如手机、平板等），也可以用于物联网设备 |
| 服务类型 | 仅数据服务 | 数据、语音、短信（目前映翰通提供的人联网卡仅用于数据服务） |
| 连接稳定性 | 高可靠性、长时间稳定 | 高速、低延迟 |
| 应用场景 | 物联网设备（传感器、智能硬件等） | 日常通信设备 |
| 访问限制 | 只能访问特定的地址 | 访问一般不受限制 |
| 使用限制 | 参考【物联网卡使用限制】 | 必须实名认证 |

> **物联网卡使用限制**
>
> - **定向流量**：物联网定向流量只允许用户终端访问客户预先设置的业务平台或应用系统，不允许访问任何其他网络和服务。
> - **无法访问人联网设备或应用**：物联网卡默认是企业实名制的，只能用于物联网设备，不能用于人联网设备（包括但不限于手机等）和访问人联网应用，如社交类（微信、QQ、微博等）、购物类（淘宝、京东等）、视频类（抖音、快手、爱奇艺等）、游戏类（QQ游戏等）。
> - **机卡绑定**：物联网卡统一设置了"机卡绑定"功能，当卡首次插入设备A激活成功时，即和设备A自动绑定。如果物联网卡被换到了设备B，将触发机卡分离停机，需要机卡解绑/重绑，才能在设备B上正常使用。
> - **达量断网**：单卡当月已用流量超出套餐额度则自动断网，次月自动恢复；流量池内所有卡当月已用流量总和超出流量池额度总和则自动断网。
> - **区域使用限制**：物联网实体卡仅限中国内地使用（新疆、西藏、广西、云南边境等高风险区域除外），不能在海外区域使用。
> - **不提供语音和短信功能**：物联网卡只提供流量功能，不提供语音和短信功能。
> - **不支持更换运营商**：下单后不支持更换运营商。
> - **不支持指定号段**。
> - 不能放入手机使用，可能被运营商判定为异常导致停机。

### 1.3.2 单网卡与多网卡的区别

| 特性 | 单网卡 | 多网卡 |
|------|--------|--------|
| 网络连接数 | 仅通过一张网卡连接 | 通过两张或更多网卡连接多个网络 |
| 冗余与容错 | 无冗余，一旦失效无法连接 | 提供冗余和容错能力，网络失败时可切换 |
| 带宽与性能 | 满足基本的网络连接和数据传输需求 | 提供更稳定、更快速的网络连接，满足高并发、大数据量传输等需求 |
| 管理与配置 | 无需特殊配置 | 使用多网卡的设备需适配多网卡SDK，可远程配置切卡策略 |
| 成本 | 较低，配置简单 | 较高，需要更多配置和管理 |
| 适用场景 | 对网络性能要求不高的应用/场景，如手机、普通路由器等 | 需要高可靠性和高带宽的应用/场景，如企业级路由器 |

物联网卡的尺寸如下，单位为毫米：

- 三切卡：指同时支持大卡、Micro卡和Nano卡三种尺寸的物联网卡。即客户不需自行裁剪，可以用三种不同的方式把物联网卡从卡槽中掰下来。

#### 卡号说明

- **ICCID**：物联网卡的卡号，用于InLink平台对物联网卡的管理，比如查询卡信息、充值加油包、续费套餐等。
  - 移动卡：ICCID为20位字符，与实体卡上的号码一致；
  - 联通卡：ICCID为20位数字。实体物联网上的卡号最后一位为字母，卡添加到InLink平台上时，系统将自动转换成数字，因此在InLink平台管理物联网卡时，需使用平台上的ICCID。
  - 电信卡：ICCID为19位字符，与实体卡上的号码一致。
- **多网卡ID**：多网卡的卡号，为9位字符。在InLink平台对物联网卡的管理，比如查询卡信息、充值加油包、续费套餐等，都需通过多网卡ID进行。

## 2.2 三步快速入门流程

### 第一步：采购SIM卡

1. 联系销售咨询采购。
2. 购卡前先在[InLink平台](https://link.inhandcloud.cn/)注册账号，并在订单里备注账号名称。
3. 购卡时明确需求，包括流量套餐、SIM卡数量、定向地址清单、是否组流量池。

**验证**：采购完成后，SIM卡将发货至指定地址。

### 第二步：登录InLink平台

1. 使用浏览器访问InLink平台地址：<https://link.inhandcloud.cn/>。
2. 输入账号和密码，点击"登录"按钮。

<p align="center"><img src="./img/I1yPoNd-hV36sN7O/1735869707780-5fabeb19-7f26-4e17-98b6-27598a56362b-943189.png" alt="注册登录页面"></p>

<p align="center"><strong>图 2-1 注册登录页面</strong></p>

**验证**：成功进入InLink平台控制台页面。

### 第三步：查看SIM卡

1. 采购发货后，使用采购时提供的平台账号登录InLink平台，即可看到SIM卡。
2. 进入【SIM卡管理】页面，查看SIM卡状态和基本信息。

**验证**：SIM卡出现在列表中，状态显示正常。

## 2.3 常见入门问题排查

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 无法登录平台 | 账号密码错误 | 确认账号密码正确，或通过"忘记密码"重置 |
| 无法看到SIM卡 | 账号不一致 | 确认登录账号与购卡时备注的账号名称一致 |
| SIM卡显示未激活 | 未购买套餐 | 在"充值续费"页面购买套餐开通服务 |
| 卡插入设备无法上网 | 定向地址未配置 | 检查需要访问的地址是否已在白名单中 |

# 第4章 SIM卡接入与管理

## 4.1 SIM卡类型说明

InLink平台支持管理的SIM卡类型包括单网卡和多网卡。

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| 单网卡 | 内置一个运营商芯片，只能使用该运营商的服务 | 固定运营商网络、成本敏感场景 |
| 多网卡 | 内置多个运营商芯片，可切换使用中国移动、中国联通、中国电信服务 | 需要网络冗余和高可靠性的场景 |

## 4.2 SIM卡添加与激活流程

### 采购SIM卡

1. 联系销售咨询采购；
2. 在InLink平台注册账号，并在订单里备注账号名称；
3. 明确需求：流量套餐、SIM卡数量、定向地址清单、是否组流量池。

### SIM卡激活

采购发货后，使用采购时提供的平台账号登录InLink平台，即可看到SIM卡。

SIM卡激活方式：

- **测试期到期强制激活**：测试期默认3个月，超过3个月卡自动激活，进入计费。
- **测试期流量用完强制激活**：测试期流量默认2MB（单网卡）/30MB（多网卡），用完自动激活，进入计费。
- **手动开通服务**：对于"未激活"状态的白卡，可在"充值续费"页面购买套餐开通服务。

### 5.4.3 流量池扩池

当流量池的总量用尽后，流量池中的卡将被停机。可通过以下方式复机：

- 单卡充值流量包：为流量池中的卡充值流量包后，流量池将被重新激活使用；
- 添加卡到流量池：向流量池中添加SIM卡进行扩池。

如需向流量池中添加卡或移出卡，请联系客户经理。

## 附录 安全注意事项

1. 账号密码应定期更换，避免使用简单密码。
2. 离开工位时应退出平台登录状态。
3. 不要将账号密码告知他人。
4. 物联网卡只能用于物联网设备，不能用于手机等人联网设备。
5. 物联网卡不支持更换运营商，下单前需确认需求。
6. 物联网实体卡仅限中国内地使用，不能在海外区域使用。
7. 定向地址变更需3-15个工作日，请合理规划访问地址。

> **警告**：管理员账号具有最高权限，请妥善保管。

## FAQ 常见问题解答

### 登录类问题

**问题1：忘记密码怎么办？**

1. 在登录页面点击"忘记密码"。
2. 输入注册邮箱或手机号。
3. 按提示完成验证，设置新密码。

### SIM卡使用类问题

**问题2：卡插入设备后为什么无法上网？**

1. 检查卡是否正常安装：插卡后设备的蜂窝信号灯是否正常；
2. 检查卡的状态是否正常：在平台查看卡是否已停机，或将SIM卡插入其他设备测试；
3. 检查设备是否正常：插入其他正常工作的SIM卡后查看设备是否可用；
4. 检查定向地址是否正确：如卡未配置需要访问的地址，可在平台提交定向申请；
5. 如以上均核实无问题，请联系客户经理。

**问题3："未激活"状态的卡怎么激活？**

在"业务办理与计费 >> 充值续费 >> 开通服务"为卡购买套餐后，系统将自动激活卡。

**问题4：卡的测试期是什么？什么时候结束？**

卡的测试期为3个月，期间共有2MB（单网卡）/30MB（多网卡）的流量供测试使用。测试期从向映翰通下单之日起算。当测试期流量超过限额或测试期结束后，卡将被自动激活，开始正式计费。

**问题5：卡换绑设备后为什么停机了？**

工信部对物联网卡的使用有严格限制，物联网设备的IMEI需要与物联网卡的IMSI一一对应。卡首次插入设备后，就与设备机卡绑定，原则上不允许解绑。如因设备损坏等原因确需解绑，请联系客户经理。

**问题6：怎么更改卡的白名单地址？**

所有SIM卡默认包含6个定向地址。如需修改，可在InLink平台的"业务办理与计费 >> 业务办理"发起定向申请。

**问题7：SIM卡列表的信息是实时更新的吗？**

由于运营商侧限制，SIM卡列表不支持实时更新。如需查看卡的实时数据，可在卡详情手动刷新。

**问题8：卡添加报错怎么办？**

如遇添加SIM卡报错，请确认卡号是否正确、是否其他人员已添加过。如以上都不是，请联系客户经理。

- 方式2：配置切卡策略：当卡的信号满足配置条件后，可自动触发切卡。

### 人联网卡实名问题

**问题10：人联网卡如何实名？**

如所购的卡为人联网卡，需先在运营商实名小程序上进行实名，实名成功后才能正常使用。

实名前准备材料：本人身份证、卡的ICCID和MSISDN。

运营商实名小程序：
- 中国联通：搜索"联通物联网卡实名认证系统"小程序。
- 中国移动：搜索"中移物联卡实名登记"小程序。
- 中国电信：搜索"天翼物联"服务号，进入"一站服务>>实名认证"。

### 流量池相关问题

**问题11：什么是流量池？**

同一套餐规格的月结卡可以组成一个流量池，对池内所有卡进行统一计费。池内卡流量共享，一张卡的流量用尽后可使用另一张卡的流量。流量池总流量用尽后，池内卡才会因流量用尽而停机。

**问题12：怎么为流量池扩池？**

- 单卡充值加油包：为流量池中的卡充值加油包后，流量池将重新激活使用；
- 向池内添加卡：可向流量池中添加卡进行扩池。如需组建流量池，请联系客户经理。

**问题13：流量池流量不足时有通知吗？**

可到"策略与规则 >> 自动化规则"配置流量池流量预警，支持邮件和短信通知。

### 充值续费相关问题

**问题14：卡的套餐到期或流量用尽怎么办？**

- 套餐到期：可在平台套餐续费；
- 流量用尽：月卡当月流量用尽后，可充值加油包重新激活使用。

**问题15：充值后卡的套餐什么时候生效？**

- 续费套餐：续费的套餐将在当前套餐用尽后开始生效；
- 加油包充值：加油包购买后立即生效。

中国移动的卡可能存在充值延迟，如遇到充值后无法立即生效，请在30分钟后再观察。如卡仍未恢复，请联系客户经理或在平台提交反馈。

**问题16：联通卡充值续费时提示"SIM卡不存在"？**

如卡在账号下仍遇到此报错，请确认输入的ICCID是否在InLink的SIM卡列表存在。联通实体卡片上展示的卡号为加密卡号，仅用于添加卡到InLink平台。后续在InLink平台上对卡的管理操作，需使用添加到InLink后的真实卡号。
