## 最短上手路径

若目标为尽快看到桌面或能通过 SSH 登录，可按下列顺序操作（细节见后文对应章节）。

1. **准备**：USB Type-C 电源（5 V／5 A）、≥ 16 GB 且 Class 10（或更快）的 Micro SD 卡、PWM 风扇（4-pin、5 V）、已下载的 MO68A Armbian 镜像、RJ45 网线；（桌面路径另备 Mini DisplayPort 线缆、DisplayPort 显示器、USB 键盘与鼠标）。
2. **烧录**：在主机上使用 Armbian Imager 将镜像写入 SD 卡；写入会清空该卡全部数据。
3. **接线**：先接风扇、插入 SD 卡、按需连接显示器与 USB 外设与网线，再接串口调试线（可选）；**电源最后接 USB-C（J5）**。
4. **上电**：LED（D1）应先呈红色；系统就绪后变为绿色常亮。
5. **等待**：首次启动会因根分区扩容额外耗时约 1～2 分钟并自动重启一次；之后常规冷启动约 **30～45 秒**。
6. **登录**：首次使用默认账户用户名 `root`，密码 `1234`；完成 Armbian 一次性初始化向导（语言、时区、新建用户等）。
7. **远程**（无头路径）：在局域网内查到板卡 IP 后执行 `ssh <用户名>@<板卡IP>`（参见 [网络访问](#网络访问)）。

## 设备概览

### 关于 MO68A

MO68A 是一款基于 **TDA4VE / AM68A SoC**（TI J721S2 系列）的紧凑型 AI 边缘计算板卡，具备双核 Cortex-A72 应用处理器与 **8 TOPS**（MMA）AI 加速能力，兼容树莓派 **40-pin HAT**。板卡运行 **Armbian Linux（Debian 13 Trixie）**，可作为连接显示器与外设的桌面计算机使用，也可在无显示器情况下通过串口或 SSH 运维。

### 主要规格

| 参数 | 规格 |
| --- | --- |
| SoC | TDA4VE / AM68A（TI J721S2） |
| CPU | 2× ARM Cortex-A72，最高 2.0 GHz |
| AI 加速器 | MMA，8 TOPS |
| 内存 | 8 GB LPDDR4 |
| 存储 | Micro SD（主启动）；PCIe NVMe（扩展，见用户指南） |
| 显示 | Mini DisplayPort |
| 网络 | 1× 千兆以太网 RJ45 |
| USB | 4× USB 3.0 Type-A |
| 摄像头 / 显示扩展 | 2× MIPI 22-pin FPC（CSI-2 / DSI） |
| 扩展接口 | 40-pin GPIO（兼容树莓派 HAT） |
| 供电 | USB-C，5 V / 5 A（27 W） |
| 操作系统 | Armbian（Debian 13 Trixie），XFCE 桌面 |

## 准备物品

### 必备

| 物品 | 说明 |
| --- | --- |
| MO68A 板卡 | — |
| USB-C 电源适配器 | 5 V / 5 A（25～27 W） |
| Micro SD 卡 | ≥ 16 GB，Class 10 / UHS-I（U1）或更快 |
| PWM 风扇 | 5 V，4-pin |
| RJ45 网线 | 用于有线网络（查 IP、SSH） |
| 主机电脑 | Windows、macOS 或 Linux，用于烧录 SD 卡 |

### 可选（建议按场景准备）

| 物品 | 说明 |
| --- | --- |
| Mini DisplayPort 线缆、DisplayPort 显示器 | **桌面路径**必备 |
| USB 键盘和鼠标 | **桌面路径**操作 XFCE |
| USB 转 UART 适配器（3.3 V） | **无头路径**排障或串口完成初始化 |
| RTC 电池（CR2032，2-pin JST-SH 等） | 持久时钟（详见用户指南 §11） |

**路径说明**：**桌面路径** — 连接显示器、键盘和鼠标，启动后进入 XFCE。**无头路径** — 主要依赖网线（可选串口），通过 SSH 访问；烧录与接线的前期步骤相同，登录方式见 [首次启动](#首次启动) 与 [网络访问](#网络访问)。

## 烧录 SD 卡

### 下载镜像

须从**映翰通官网**下载适用于 MO68A 的 Armbian 镜像文件，保存至用于烧录的主机电脑。

### 使用 Armbian Imager 烧录

1. 从 [Armbian Imager 发行页](https://github.com/armbian/imager/releases) 下载并安装 **Armbian Imager**。
2. 将 Micro SD 卡插入主机电脑。
3. 打开 Armbian Imager，点击 **Use Custom Image**，选择上一步下载的镜像文件。
   
   ![Use Custom Image](images/imager_custom_image.png)

4. 点击 **CHOOSE STORAGE**，选择对应的 Micro SD 卡。
   
   ![Choose Storage](images/imager_choose_storage.png)

   > **警告：** 须仔细核对所选存储设备；烧录将清除该设备上的全部已有数据。

5. 点击 **Erase & Flash**。过程中可能提示输入管理员密码；烧录与校验约需 3～5 分钟。
   
   ![Erase & Flash confirmation](images/imager_erase_flash.png)

6. 完成后在系统中「安全弹出」存储设备再拔卡。

## 硬件连接

建议顺序如下（可降低带电插拔风险；**电源始终最后连接**）。实物接线可参考下图。

![MO68A 接线实物图](images/MO68A_setup.jpg)

| 步骤 | 连接 | 连接器 | 确认 |
| --- | --- | --- | --- |
| 1 | 风扇（4-pin） | J7 | 上电后风扇转动 |
| 2 | 已烧录的 Micro SD 卡 | J23 | 插入到位（咔哒声） |
| 3 | Mini DisplayPort 线缆（*桌面路径*） | J9 | 启动前显示器可为「无信号」 |
| 4 | USB 键盘和鼠标（*桌面路径*） | J2 或 J3 | — |
| 5 | RJ45 网线 | J1 | — |
| 6 | 串口调试线缆（*可选*） | J6 | 见下文「串口调试控制台」 |
| 7 | USB-C 电源 | J5 | 上电后 D1 亮红灯；就绪后为绿灯常亮 |

### 串口调试控制台（可选）

将 **3.3 V** USB 转 UART 适配器连接至 **J6**（SH1.0 3-pin）：

| 引脚 | 信号 |
| --- | --- |
| 1 | RXD |
| 2 | GND |
| 3 | TXD |

参数：**115200 波特率，8N1，无流控**

```bash
# Linux 主机示例（将设备节点换为实际路径）
minicom -D /dev/ttyUSB0 -b 115200
```

Windows 可使用 PuTTY、Tera Term 等，选择对应 COM 端口。

串口可显示完整启动日志并提供与显示器无关的登录终端；建议在接通电源前打开串口终端，以免错过早期输出。

## 首次启动

### 启动过程

上电后系统从 Micro SD 启动。常规冷启动约 **30～45 秒**；LED（D1）变为**绿色常亮**表示系统已就绪（启动过程中可为红色）。

> **首次启动说明：** 第一次上电时，根文件系统将扩展至 SD 卡可用容量，该过程约 **1～2 分钟**，结束后板卡自动重启。须在**第二次启动完成**后再进行登录。初始化向导与桌面详见下文。

### 首次登录与初始化向导

使用默认凭据开始初始化：**用户名 `root`，密码 `1234`**。

首次通过显示器或串口登录时，Armbian 会运行一次性向导，用于设置语言、时区并创建用户账户，按界面提示完成即可。

> **提示：** 启动前连接网线时，Armbian 可根据网络环境推荐合适的语言区域设置。

![Armbian 首次登录初始化向导](images/firstlogin_cn.png)

### 桌面路径

向导完成后，XFCE 图形桌面显示在显示器上。

![Armbian XFCE 桌面](images/desktop_cn.png)

### 无头路径

通过串口完成向导后进入命令行界面。

![串口登录后的 Armbian 命令行](images/serial_login.png)

远程 SSH 步骤见 [网络访问](#网络访问)。

## 网络访问

### 有线以太网

板载千兆口（J1）连接网线后，`eth0` 默认通过 DHCP 获取 IPv4。

### 查找板卡 IP

- 在路由器管理界面查看 DHCP 客户端列表；或  
- 在板卡已登录的终端（显示器、串口或已有会话）执行：

```bash
ip addr show eth0
```

### SSH 登录

在局域网内的另一台电脑上执行：

```bash
ssh <用户名>@<板卡IP>
```

其中 `<用户名>` 为初始化向导中创建的用户（或仍使用 `root`，依实际安全配置而定）。

## 常见问题与技巧

| 现象 | 建议检查 | 更多信息 |
| --- | --- | --- |
| **无显示输出** | 显示器须支持 DisplayPort 输入（非仅 HDMI）；Mini DP 线缆接牢 J9；显示器输入源正确 | [用户指南 §7](MO68A_用户指南_V1.0.md#7-mini-displayport) |
| **无法从 SD 卡启动** | SD 卡完全插入 J23（咔哒声）；使用 Armbian Imager 重新烧录并安全弹出后再使用 | [本文「烧录 SD 卡」](#烧录-sd-卡) |
| **电源 LED 不亮** | 适配器是否为 5 V / 5 A；线缆接在 J5（USB-C 电源） | [硬件参考手册 — 电源供应](MO68A_硬件参考手册_V1.0.md#电源供应) |
| **局域网中找不到板卡** | RJ45 接 J1；交换机 / 路由器链路指示灯；在 DHCP 客户端列表中查找 IP | 本文 §网络访问 |
