## 1. 简介

白鹰能源管家专为光伏储能管理、配电房监控、生产能耗监测和办公空间节能等关键领域设计。该平台提供客制化的关键指标看板、高效的节能策略设置、直观的 WEB 组态界面、深度的能效分析、全面的运维管理以及灵活的权限管理。搭配映翰通能源网关等智能硬件，可帮助企业实现能耗监控和分析，助力节能减排，确保设备安全可靠运行。

## 2. 名词解释

| 术语 | 说明 |
|------|------|
| 租户 | 一个独立的企业，每个租户之间的设备、用户等数据完全隔离 |
| 组织 | 通常代表企业中的子公司、部门等单位，用于区分每个组织具备的监控设备等数据权限 |
| 站点组织 | 一种特殊的组织类型，除具备一般组织的数据权限划分功能外，还具备站点属性，支持配置站点电价等操作 |
| 站点 | 一个站点组织对应一个站点，通常表示一个工厂、一栋建筑或其他能源管理单元 |
| 通信设备 | 实现数据采集、数据处理与数据上云的通信网关设备，用于连接监控设备与白鹰平台 |
| 监控设备 | 需要在平台中监控的各类设备，如空调、电表、照明设备等 |
| 变量 | 监控设备中需要监控的具体数据，如电压、电流、温度、湿度等 |
| 变量类型 | 变量的分类，平台内置电压、电流、功率因数等类型，用于统计分析和数据管理 |
| 设备类型 | 监控设备的分类，如变压器、水表、电表等 |
| 管理维度 | 用于能效管理的各种维度，如区域、部门、楼层等 |
| 节点 | 管理维度中的每个具体单位，如区域管理维度中的生产区 |
| 全局角色 | 所有组织均可以查看和分配的角色 |
| 组织角色 | 特定组织才能查看和分配的角色 |
| 计费线路 | 用于设置哪些监控设备的用能数据作为站点的用能统计数据，用于计算站点每月的电费 |
| 分时电价 | 按高峰用电和低谷用电分别计算电费的电价制度 |
| 不分时电价 | 不区分高峰和低谷用电，电费按统一标准计算的电价制度 |
| DeviceSupervisor Agent（DSA） | 运行在通信设备中实现数据采集、处理和上云的数据采集软件 |
| 控制器 | 被采集数据的设备，如电表、PLC、逆变器等 |
| 测点 | 设备的业务数据，如 A 相电流、A 相电压等 |

## 3. 快速入门

以下以 EC322 为例，介绍如何配置网关联网、采集电表数据并上传至云端，最终实现云端监控电表数据。详细的 DSA 配置操作，请参考 [DSA 用户手册](https://help.inhand.com.cn/portal/zh/kb/articles/dsa%E4%B8%AD%E6%96%87)。

### 3.1 设备联网

要实现云端监控，需确保设备已联网。不同产品系列的配置方式存在差异，请参考同系列产品进行配置。各设备因固件版本不同，界面可能存在差异，以实际设备为准。

**方式一：以太网联网**

环境：用网线将 PC 与 EC322 的 ETH1（默认 192.168.3.100）接口连接，ETH2（默认 192.168.4.100）接口连接 Internet。

步骤一：为 PC 配置 192.168.3.X 网段的 IP 地址（如 192.168.3.110），掩码 255.255.255.0。在浏览器中输入 <https://192.168.3.100:9100> 访问设备页面。

![1717739348372-d3aeeb8d-9b35-4ea1-9934-3cf61b14b3cd.png](./img/RPTlnNabsvN7n1k6/1717739348372-d3aeeb8d-9b35-4ea1-9934-3cf61b14b3cd-052034.png)

步骤二：进入"系统 〉 网络管理 〉 接口"页面，为 ETH2 配置 DHCP 或静态 IP。

![1717739348604-a823b214-6322-4760-868f-275683708065.png](./img/RPTlnNabsvN7n1k6/1717739348604-a823b214-6322-4760-868f-275683708065-924807.png)

步骤三：进入"系统 〉 网络管理 〉 静态路由"，添加相应的静态路由完成联网。

![1717739348864-b2226af7-f476-420d-a940-cd5d05a5db40.png](./img/RPTlnNabsvN7n1k6/1717739348864-b2226af7-f476-420d-a940-cd5d05a5db40-935781.png)

**方式二：蜂窝联网**

一般情况下，SIM 卡插卡即可联网。如有定制需求，可在"系统 〉 网络管理 〉 蜂窝网络"页面配置 APN 等参数。

![1717739349089-41f04c2e-6be5-4dac-9997-ead349106e68.png](./img/RPTlnNabsvN7n1k6/1717739349089-41f04c2e-6be5-4dac-9997-ead349106e68-256457.png)

**方式三：Wi-Fi（STA）联网**

在"系统 〉 网络管理 〉 WiFi"菜单下点击"启用 WiFi"，配置相关参数，使其连接到现场的 SSID 实现联网。

![1717739349290-fd548a9e-a5d2-4ae0-87eb-a2fb1eb9c080.png](./img/RPTlnNabsvN7n1k6/1717739349290-fd548a9e-a5d2-4ae0-87eb-a2fb1eb9c080-000293.png)

### 3.2 数采上云配置

#### 3.2.1 管理 DSA

EC322 出厂默认安装了 DSA。在"系统 〉 插件"页面，可点击"添加"按钮并选择新的 DSA 版本文件以更新 DSA 版本（版本在官网的资源中心获取）。可点击"暂停"或"启动"按钮以启动或暂停 DSA。

![1717739349488-3a939a99-675a-48b7-a3a6-01406db325cf.png](./img/RPTlnNabsvN7n1k6/1717739349488-3a939a99-675a-48b7-a3a6-01406db325cf-043107.png)

#### 3.2.2 数据采集配置

进入"设备监控 〉 测点监控 〉 监控列表"页面。在该界面可配置和监控控制器以及测点状态。

步骤一：点击控制器列表的"添加"按钮，选择通信协议并配置相应的通信参数。

![1717739349702-ee35203d-85f5-4b2f-b59b-25ca6f0f2856.png](./img/RPTlnNabsvN7n1k6/1717739349702-ee35203d-85f5-4b2f-b59b-25ca6f0f2856-663205.png)

步骤二：控制器添加完成后，点击测点列表的"添加"按钮，配置相应的采集信息。

![1717739349892-06367582-1ba1-4546-bc17-6ebadb9e95d3.png](./img/RPTlnNabsvN7n1k6/1717739349892-06367582-1ba1-4546-bc17-6ebadb9e95d3-820349.png)

步骤三：配置完成后可在该页面监控控制器和测点的状态。

![1717739350149-9245030f-f6ff-43c6-b263-4231d172f85c.png](./img/RPTlnNabsvN7n1k6/1717739350149-9245030f-f6ff-43c6-b263-4231d172f85c-825244.png)

#### 3.2.3 数据上云配置

进入"设备监控 〉 云服务"页面。在该界面可配置测点数据传输到白鹰能源管家。

启用云服务后，选择云平台类型为"白鹰能源管家"，选择服务器地址并点击"提交"即可。成功连接平台后，云服务状态为"连接成功"。

![1717739350388-f93de605-87b3-4bac-9e45-c30deab5f9c0.png](./img/RPTlnNabsvN7n1k6/1717739350388-f93de605-87b3-4bac-9e45-c30deab5f9c0-362562.png)

### 3.3 云平台监控

#### 3.3.1 添加通信设备

进入"系统管理 〉 设备管理 〉 监控设备管理"页面。点击"添加"按钮，配置并添加一台监控设备。

点击该设备的"变量管理"按钮，进入"变量管理"页面。

![1717739351537-bd147897-e227-43bd-b1c5-bb32ff2e6eab.png](./img/RPTlnNabsvN7n1k6/1717739351537-bd147897-e227-43bd-b1c5-bb32ff2e6eab-095628.png)

点击"新增"按钮，添加需要在云平台中监控的变量。

![1717739351855-af201c9f-1800-4848-bac7-6b5ddcde1beb.png](./img/RPTlnNabsvN7n1k6/1717739351855-af201c9f-1800-4848-bac7-6b5ddcde1beb-295854.png)

配置完成后，需点击"应用"按钮使新的监控设备配置生效。

![1717739352061-5f0c9249-4150-436f-9dbb-3311fb520372.png](./img/RPTlnNabsvN7n1k6/1717739352061-5f0c9249-4150-436f-9dbb-3311fb520372-616830.png)

白鹰能源管家的地址为 xenergy.inhandcloud.cn。用户可点击"立即注册"注册一个新租户（即一个独立的企业），该租户下的设备、用户等数据与其他租户完全隔离。注册完成后，输入相应的登录信息点击"登录"即可。

![1717739352807-6a60bf3c-362f-4550-9881-5a2bd2b97634.png](./img/RPTlnNabsvN7n1k6/1717739352807-6a60bf3c-362f-4550-9881-5a2bd2b97634-875090.png)

#### 4.2.2 站点大屏

站点大屏页面支持直观展示站点状态信息。展示的信息包括：站点信息（电压等级、变压器装机容量、供电线路）；设备统计（通信设备、监控设备、变量点）；用能趋势（水电气年月日维度）；场站图（推荐图片比例 3:2）；负荷趋势（实时负荷及 0~24 小时今日与昨日负荷趋势）；能耗统计（本月和本年总能耗）；用能统计（水电气年月日维度环比）；用电排名（TOP3 能源节点，不统计根能源节点）。

![1717739355421-ff0e7f64-4100-4535-b54e-5c605a9b4172.png](./img/RPTlnNabsvN7n1k6/1717739355421-ff0e7f64-4100-4535-b54e-5c605a9b4172-923091.png)

### 4.3 安全运行

#### 4.3.1 设备监控

设备监控页面提供单个监控设备详细的状态展示和查询。在"系统管理 〉 监控设备管理"添加设备后，用户可在此页面查看监控设备的实时状态、历史趋势和告警记录等信息。左侧的设备列表可通过组织、标签和设备名称筛选过滤。

**概览：** 可快速掌握监控设备的运行情况，包括实时运行状态、历史运行趋势、用能和尖峰平谷用电占比。

![1717739355732-f99e4d5b-80b5-4b87-8bee-3b0366d40ec0.png](./img/RPTlnNabsvN7n1k6/1717739355732-f99e4d5b-80b5-4b87-8bee-3b0366d40ec0-556224.png)

![1717739356351-f8143569-0b57-497b-937a-5a09f5943785.png](./img/RPTlnNabsvN7n1k6/1717739356351-f8143569-0b57-497b-937a-5a09f5943785-346869.png)

**数据报表：** 可查看变量的历史数据报表。可根据需求自定义查看的变量类型和数据时间范围。

**告警记录：** 可查看当前监控设备的告警信息。可自定义查看的告警发生时间范围。

#### 4.3.2 WEB 组态

WEB 组态页面支持用户自定义 WEB 组态画面，提供丰富的常用组件和电力组件，帮助用户通过组态画面直观展示关键场所的设备运行状态。

**新建视频设备：** 在"视频配置"页面，点击"新建视频设备"按钮，配置相关参数点击"确定"完成添加。

**通道管理：** 点击视频设备的"通道管理"按钮，进入"通道管理"页面。点击"新建通道"，配置通道名称和通道号完成通道添加。通道添加完成后即可实时监控视频画面了。
